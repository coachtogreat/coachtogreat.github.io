Daftar Berurutan (Ordered List) di HTML & CSS

Daftar berurutan (Ordered List) adalah elemen HTML yang digunakan untuk menampilkan daftar item secara berurutan dengan nomor atau huruf. Berbeda dengan daftar tidak berurutan yang menggunakan bullet, daftar berurutan menggunakan penomoran untuk menandai setiap item.

Konsep Dasar Ordered List:

  • <ol>: Tag pembungkus untuk membuat daftar berurutan (ordered list)
  • <li>: Tag untuk setiap item dalam daftar (list item)
  • Atribut type: Menentukan jenis penomoran (1, A, a, I, i)
  • Atribut start: Menentukan nomor awal penomoran
  • Atribut reversed: Membalik urutan penomoran

1. Daftar Berurutan Dasar

Berikut adalah contoh sederhana penggunaan ordered list dengan penomoran default (angka):

HTML 
<ol>
  <li>Bangun tidur</li>
  <li>Mandi pagi</li>
  <li>Sarapan</li>
  <li>Berangkat kerja</li>
</ol>

Hasil:

  1. Bangun tidur
  2. Mandi pagi
  3. Sarapan
  4. Berangkat kerja

2. Jenis Penomoran

Anda dapat mengubah jenis penomoran menggunakan atribut type:

type:

Atribut Tambahan:

HTML 
<ol type="1">
  <li>Item pertama</li>
  <li>Item kedua</li>
  <li>Item ketiga</li>
</ol>

Hasil:

  1. Item pertama
  2. Item kedua
  3. Item ketiga

3. Styling dengan CSS

Anda dapat mengubah tampilan daftar berurutan menggunakan CSS:

CSS 
.custom-list {
  list-style-type: none;
  counter-reset: my-counter;
  padding-left: 0;
}

.custom-list li {
  position: relative;
  padding-left: 40px;
  margin-bottom: 15px;
  line-height: 30px;
  background-color: #f5f5f5;
  border-radius: 4px;
  padding: 10px 10px 10px 50px;
}

.custom-list li::before {
  counter-increment: my-counter;
  content: counter(my-counter);
  position: absolute;
  left: 0;
  top: 50%;
  transform: translateY(-50%);
  width: 30px;
  height: 30px;
  background-color: #3498db;
  color: white;
  border-radius: 50%;
  display: flex;
  align-items: center;
  justify-content: center;
  font-weight: bold;
  font-size: 14px;
}

Hasil:

  1. Pertama, siapkan bahan-bahan
  2. Kedua, campurkan semua bahan
  3. Ketiga, aduk hingga rata
  4. Keempat, masak dengan api sedang
  5. Kelima, sajikan selagi hangat

4. Daftar Bersarang (Nested List)

Anda dapat membuat daftar di dalam daftar untuk membuat hierarki yang lebih kompleks:

HTML 
<ol>
  <li>Bab 1: Pengenalan
    <ol type="a">
      <li>1.1 Pendahuluan</li>
      <li>1.2 Tujuan</li>
    </ol>
  </li>
  <li>Bab 2: Pembahasan
    <ol type="a">
      <li>2.1 Metodologi
        <ol type="i">
          <li>2.1.1 Alat</li>
          <li>2.1.2 Bahan</li>
        </ol>
      </li>
      <li>2.2 Hasil</li>
    </ol>
  </li>
  <li>Bab 3: Penutup</li>
</ol>

Hasil:

  1. Bab 1: Pengenalan
    1. 1.1 Pendahuluan
    2. 1.2 Tujuan
  2. Bab 2: Pembahasan
    1. 2.1 Metodologi
      1. 2.1.1 Alat
      2. 2.1.2 Bahan
    2. 2.2 Hasil
  3. Bab 3: Penutup
