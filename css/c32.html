Mengatur Tampilan Teks di CSS

Pengaturan teks yang baik sangat penting untuk keterbacaan dan estetika website. Di bab ini, kita akan mempelajari properti CSS dasar untuk mengatur tampilan teks, termasuk font-size, font-family, text-align, dan line-height.

1. Mengatur Ukuran Font (font-size)

Properti font-size digunakan untuk mengatur ukuran teks. Berikut beberapa cara untuk menentukan ukuran font:

CSS 
/* Menggunakan satuan piksel */
h1 {
  font-size: 32px;
}

/* Menggunakan satuan em (relatif terhadap ukuran font parent) */
p {
  font-size: 1em;
}

/* Menggunakan satuan rem (relatif terhadap ukuran font root/html) */
body {
  font-size: 1rem;
}

/* Menggunakan satuan viewport width (responsive) */
.responsive-text {
  font-size: 2vw;
}

/* Menggunakan kata kunci */
.small-text {
  font-size: small;
}
.large-text {
  font-size: large;
}
.xlarge-text {
  font-size: x-large;
}

Satuan yang Umum Digunakan:

  • px - Piksel (1px = 1/96 inci)
  • em - Relatif terhadap ukuran font parent
  • rem - Relatif terhadap ukuran font root (html)
  • % - Persentase dari ukuran font parent
  • vw - 1% dari lebar viewport

2. Memilih Jenis Huruf (font-family)

Properti font-family memungkinkan Anda menentukan jenis huruf yang akan digunakan. Selalu sediakan beberapa alternatif font (font stack) untuk kompatibilitas.

CSS 
/* Contoh font stack */
body {
  font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
}

/* Menggunakan Google Fonts */
@import url('https://fonts.googleapis.com/css2?family=Poppins:wght@400;700&display=swap');

.heading {
  font-family: 'Poppins', sans-serif;
  font-weight: 700;
}

/* Menggunakan font lokal */
@font-face {
  font-family: 'CustomFont';
  src: url('fonts/custom-font.woff2') format('woff2');
  font-weight: normal;
  font-style: normal;
}

.custom-text {
  font-family: 'CustomFont', Arial, sans-serif;
}
Contoh teks dengan font stack: Segoe UI, Tahoma, Geneva, Verdana, sans-serif
Contoh teks dengan font monospace: Courier New, Courier, monospace

3. Perataan Teks (text-align)

Properti text-align mengatur perataan teks secara horizontal dalam elemen blok.

CSS 
/* Rata kiri (default untuk bahasa kiri ke kanan) */
.left-align {
  text-align: left;
}

/* Rata kanan */
.right-align {
  text-align: right;
}

/* Rata tengah */
.center {
  text-align: center;
}

/* Rata kiri-kanan (justify) */
.justify {
  text-align: justify;
}

4. Jarak Antar Baris (line-height)

Properti line-height mengatur jarak vertikal antara baris teks. Ini sangat penting untuk keterbacaan teks yang baik.

CSS 
/* Tanpa satuan (disarankan) - dikalikan dengan ukuran font */
.readable {
  line-height: 1.6;
}

/* Dengan satuan piksel */
.fixed-height {
  line-height: 24px;
}

/* Dengan persentase */
.percentage-height {
  line-height: 150%;
}

/* Normal (default) */
.normal {
  line-height: normal;
}

Tips untuk line-height yang baik:

  • Gunakan nilai antara 1.4 dan 1.6 untuk teks panjang (body text)
  • Judul biasanya membutuhkan line-height yang lebih kecil (1.1-1.3)
  • Lebih baik menggunakan nilai tanpa satuan untuk line-height
  • Pertimbangkan aksesibilitas - jangan membuat line-height terlalu kecil

5. Properti Teks Lainnya

CSS 
/* Ketebalan font */
.bold {
  font-weight: bold;  /* atau 700 */
}

/* Gaya font */
.italic {
  font-style: italic;
}

/* Dekorasi teks */
.underline {
  text-decoration: underline;
}

/* Jarak antar huruf */
.spaced {
  letter-spacing: 1px;
}

/* Jarak antar kata */
.word-spacing {
  word-spacing: 0.5em;
}

/* Transformasi teks */
.uppercase {
  text-transform: uppercase;
}

/* Warna teks */
.text-color {
  color: #2c3e50;
}

/* Bayangan teks */
.shadow {
  text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.3);
}

Praktik Terbaik untuk Tipografi Web:

  • Gunakan ukuran font minimal 16px untuk teks utama
  • Batasi panjang baris teks antara 50-75 karakter
  • Gunakan kontras yang cukup antara teks dan latar belakang
  • Hindari menggunakan terlalu banyak jenis font dalam satu halaman
  • Optimalkan font loading untuk performa yang lebih baik
