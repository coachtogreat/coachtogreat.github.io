Struktur Dasar HTML

1. Apa itu Tag HTML?

Tag HTML adalah komponen dasar dari HTML (HyperText Markup Language) yang digunakan untuk memberi struktur dan makna pada konten web. Setiap tag memiliki fungsi spesifik yang memberitahu browser bagaimana menampilkan konten tersebut.

Tag HTML dapat berupa elemen pembuka, isi, dan penutup, atau bisa juga berupa self-closing tag untuk elemen yang tidak memerlukan konten.

Struktur Dasar Tag HTML:

Tag HTML umumnya memiliki struktur sebagai berikut:

HTML 
<tagname>Konten</tagname>

Contoh:

HTML 
<p>Ini adalah paragraph.</p>

Komponen Utama Tag HTML:

  • Tag Pembuka: Diawali dengan < dan diakhiri dengan >, contoh: <p>
  • Konten: Teks atau elemen lain yang berada di antara tag pembuka dan penutup
  • Tag Penutup: Sama seperti tag pembuka tapi dengan garis miring / sebelum nama tag, contoh: </p>
  • Atribut: Informasi tambahan yang diletakkan di dalam tag pembuka, contoh: <a href="https://example.com">

2. Self-Closing Tag (Void Elements)

Beberapa elemen HTML tidak memerlukan tag penutup karena tidak memiliki konten. Elemen-elemen ini disebut void elements atau self-closing tags.

HTML 
<img src="gambar.jpg" alt="Deskripsi gambar" />
<br />
<hr />

3. Apa itu Elemen HTML?

Elemen HTML adalah komponen lengkap yang terdiri dari tag pembuka, konten (opsional), dan tag penutup. Elemen adalah blok bangunan dasar dari halaman web.

Setiap elemen dapat memiliki atribut yang memberikan informasi tambahan tentang elemen tersebut.

HTML 
<h1>Judul Utama</h1>

Itu adalah elemen <h1>.

Elemen bisa juga hanya tag tunggal (self-closing) seperti:

HTML 
<img src="assets/png/logo.png" alt="Logo" />

4. Kategori Elemen HTML Berdasarkan Fungsi

Elemen HTML dapat dikelompokkan berdasarkan fungsinya dalam membangun struktur dan tampilan halaman web.

A. Elemen Struktur Dokumen

Elemen-elemen yang membentuk kerangka dasar dokumen HTML dan mengatur struktur keseluruhan halaman.

  • <html> - Elemen utama dari dokument HTML.
  • <head> - Bagian kepala (metadata, title, link ke CSS, dll).
  • <title> - Judul halaman (muncul di tab browser).
  • <body> - Isi halaman (semua yang terlihat oleh pengguna).

B. Elemen Heading (Judul)

Digunakan untuk membuat judul dan subjudul, dengan <h1> sebagai judul utama (hanya satu per halaman untuk SEO) dan <h2> sampai <h6> untuk subjudul dengan tingkat kepentingan yang menurun.

HTML 
<h1>Judul 1</h1> sampai <h6>Judul 6</h6>

C. Elemen Teks dan Paragraf

Elemen-elemen untuk mengatur teks dan paragraf dalam dokumen HTML.

  • <p> - Paragraph.
  • <br> - Baris baru.
  • <hr> - Garis Horizontal.
  • <strong> - Teks tebal (bold).
  • <em> - Teks miring (italic).

D. Elemen Link dan Gambar

Elemen untuk menambahkan hyperlink dan gambar ke dalam halaman web.

  • <a href="url">Teks Link</a> - Hyperlink
  • <img src="url" alt="deskripsi"> - Baris baru.

E. Elemen Layout dan Struktur

Elemen-elemen yang digunakan untuk mengatur tata letak dan struktur halaman web.

  • <div> - Kontainer umum (block level)
  • <span> - Kontainer kecil (inline)
  • <section> - Bagian dokumen
  • <article> - Konten independen
  • <nav> - Navigasi
  • <header> <footer> - Bagian atas dan bawah.

F. Elemen Daftar

Digunakan untuk membuat daftar terurut (bernomor) dan tidak terurut (berpoin).

  • <ol> - Daftar beurutan (angka)
  • <ul> - Daftar tidak berurutan (bullets)
  • <li> - Item dalam daftar

G. Elemen Tabel

Digunakan untuk menampilkan data dalam bentuk tabel dengan baris dan kolom.

  • <table> - Tabel
  • <th> - Header kolom
  • <tr> - Tabel baris
  • <td> - Tabel Data

H. Elemen Formulir

Digunakan untuk membuat form interaktif yang memungkinkan pengguna memasukkan data.

  • <form> - Formulir
  • <input> - Kolom input
  • <textarea> - Area teks
  • <button> - Tombol
  • <select> dan <option> - Dropdown

I. Elemen Multimedia

Elemen untuk menambahkan konten multimedia seperti audio, video, dan canvas ke dalam halaman web.

  • <audio> - Audio
  • <video> - video
  • <source> - Sumber Media
