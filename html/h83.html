Perbandingan <div> dan <span>

Memahami Perbedaan dan Penggunaan <div> dan <span>

<div> dan <span> adalah dua elemen HTML yang sering digunakan untuk mengelompokkan dan menata konten, tetapi keduanya memiliki tujuan dan perilaku yang berbeda. Memahami perbedaannya sangat penting untuk pengembangan web yang efektif.

<div> (Division)

Elemen blok yang digunakan untuk mengelompokkan elemen-elemen lain sebagai blok konten.

Karakteristik:
  • Elemen blok (membuat baris baru sebelum dan sesudahnya)
  • Menggunakan lebar penuh yang tersedia secara default
  • Digunakan untuk mengelompokkan konten dalam blok yang lebih besar
  • Dapat berisi elemen blok dan inline lainnya
Contoh Penggunaan:
<div class="header">
  <h1>Judul Halaman</h1>
  <nav>...</nav>
</div>

<div class="content">
  <article>...</article>
  <aside>...</aside>
</div>

<span>

Elemen inline yang digunakan untuk mengelompokkan teks atau elemen inline lainnya dalam satu baris teks.

Karakteristik:
  • Elemen inline (tidak membuat baris baru)
  • Hanya mengambil ruang yang dibutuhkan oleh kontennya
  • Digunakan untuk menata sebagian teks atau elemen inline
  • Tidak dapat berisi elemen blok (secara teknis bisa, tapi tidak valid)
Contoh Penggunaan:
<p>
  Ini adalah teks biasa dengan <span class="highlight">bagian yang disorot</span> 
  dan <span style="color: red;">teks berwarna</span>.
</p>

<p>
  Harga: <span class="price">Rp 1.000.000</span> 
  <span class="discount">Diskon 10%</span>
</p>

2. Perbandingan Langsung

Aspek <div> <span>
Tipe Elemen Block-level Inline
Default Display block inline
Lebar Default 100% dari parent Sesuai konten
Margin & Padding Bisa diatur semua sisi Hanya kiri/kanan untuk margin
Penggunaan Umum Layout, sectioning, grouping elemen besar Styling teks, grouping elemen inline

4. Kapan Menggunakan Masing-masing?

Gunakan <div> ketika:
  • Membuat struktur layout halaman (header, footer, sidebar, dll)
  • Mengelompokkan beberapa elemen terkait sebagai satu unit
  • Membuat container untuk elemen blok lainnya
  • Menerapkan gaya CSS ke sekelompok elemen
Gunakan <span> ketika:
  • Menata sebagian teks dalam paragraf
  • Memberi gaya pada teks tertentu dalam konten
  • Mengelompokkan elemen inline
  • Menambahkan hook untuk JavaScript ke bagian teks

5. Praktik Terbaik

  • Gunakan elemen semantik HTML5 (seperti <header>, <nav>, <section>) alih-alih <div> bila memungkinkan
  • Hindari penggunaan <span> yang berlebihan - gunakan elemen semantik seperti <strong>, <em>, dll. jika sesuai
  • Berikan class atau id yang deskriptif untuk memudahkan styling dan JavaScript
  • Ingat bahwa <div> dan <span> seharusnya menjadi pilihan terakhir ketika tidak ada elemen semantik yang sesuai
  • Gunakan <div> untuk tata letak dan <span> untuk teks
Tip: Meskipun <div> dan <span> tidak memiliki makna semantik, mereka tetap penting untuk tata letak dan pengelompokan dalam HTML.
