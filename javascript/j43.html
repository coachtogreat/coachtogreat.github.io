Ternary Operator dalam JavaScript

1. Pengenalan Ternary Operator

Syntax Ternary Operator 
// Sintaks dasar
kondisi ? nilai_jika_benar : nilai_jika_salah;

// Contoh sederhana
let umur = 25;
let status = umur >= 18 ? "Dewasa" : "Anak-anak";
console.log(status); // "Dewasa"

2. Perbandingan dengan If-Else

Contoh Perbandingan 
// Menggunakan if-else
let nilai = 85;
let grade;
if (nilai >= 70) {
    grade = "A";
} else {
    grade = "B";
}

// Menggunakan ternary
let nilai = 85;
let grade = nilai >= 70 ? "A" : "B";

// Menggunakan if-else dengan assignment
let nilai = 85;
let grade = nilai >= 70 ? "A" : nilai >= 60 ? "B" : "C";

3. Ternary dengan Multiple Conditions

Contoh Multiple Conditions 
// Contoh grade
let nilai = 85;
let grade = nilai >= 90 ? "A" :
            nilai >= 80 ? "B" :
            nilai >= 70 ? "C" :
            nilai >= 60 ? "D" : "E";

// Contoh umur
let umur = 25;
let kategori = umur < 18 ? "Anak-anak" :
               umur < 30 ? "Dewasa muda" :
               umur < 60 ? "Dewasa" : "Lansia";

4. Ternary dengan Expression

Contoh Expression 
// Contoh dengan function
let nilai = 85;
let grade = nilai >= 70 ? hitungNilai(nilai) : "Gagal";

function hitungNilai(nilai) {
    return nilai + 5;
}

// Contoh dengan array
let angka = 5;
let hasil = angka > 0 ? [angka, angka * 2] : [0, 0];
console.log(hasil); // [5, 10]

// Contoh dengan object
let login = true;
let user = login ? { nama: "John", role: "admin" } : { nama: "Guest" };

5. Best Practices

  • Gunakan ternary untuk kondisi sederhana
  • Hindari nested ternary yang terlalu kompleks
  • Gunakan tanda kurung untuk mengklarifikasi prioritas
  • Pisahkan kondisi kompleks ke dalam fungsi
  • Gunakan if-else untuk kondisi yang lebih dari dua pilihan
  • Tambahkan komentar untuk kondisi yang rumit

6. Contoh Praktis

Contoh Validasi Form 
// Validasi email
function validateEmail(email) {
    return email.includes("@") ? "Email valid" : "Email tidak valid";
}

// Validasi umur
function validateAge(age) {
    return age >= 18 ? "Dewasa" : "Anak-anak";
}

// Contoh penggunaan
let email = "user@example.com";
let age = 25;

console.log(validateEmail(email)); // "Email valid"
console.log(validateAge(age)); // "Dewasa"

// Contoh dengan array
let angka = 5;
let hasil = angka > 0 ? [angka, angka * 2] : [0, 0];
console.log(hasil); // [5, 10]

7. Tips Penggunaan

  • Gunakan ternary untuk assignment yang sederhana
  • Gunakan if-else untuk logika yang kompleks
  • Gunakan tanda kurung untuk kondisi yang rumit
  • Tambahkan komentar untuk menjelaskan kondisi
  • Hindari nested ternary yang terlalu dalam
  • Pisahkan kode yang kompleks ke dalam fungsi
