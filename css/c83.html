Dark Mode dengan Media Query

Apa itu Dark Mode?

Dark mode adalah skema warna yang menggunakan warna gelap sebagai latar belakang dengan teks berwarna terang, memberikan kenyamanan visual di lingkungan dengan cahaya redup.

Menggunakan prefers-color-scheme

Media query prefers-color-scheme memungkinkan kita mendeteksi preferensi tema sistem pengguna: 

/* Tema default (light) */
:root {
  --bg-color: #ffffff;
  --text-color: #333333;
}

/* Tema dark */
@media (prefers-color-scheme: dark) {
  :root {
    --bg-color: #1a1a1a;
    --text-color: #f0f0f0;
  }
}

/* Menerapkan variabel */
body {
  background-color: var(--bg-color);
  color: var(--text-color);
  transition: all 0.3s ease;
}

Toggle Dark Mode dengan JavaScript

Kita bisa menambahkan toggle untuk mengubah tema secara manual:

function toggleTheme() {
  const html = document.documentElement;
  const currentTheme = html.getAttribute('data-theme');

    html.setAttribute('data-theme', 'light');
