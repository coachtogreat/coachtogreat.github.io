CSS Custom Properties Lanjutan

CSS Custom Properties (juga dikenal sebagai CSS Variables) adalah fitur CSS yang memungkinkan kita untuk mendefinisikan nilai yang dapat digunakan kembali di seluruh stylesheet. Berbeda dengan variabel CSS biasa, Custom Properties memiliki kemampuan yang lebih canggih dan fleksibel.

Mengapa Menggunakan CSS Custom Properties?

  • Dinamis - Nilai dapat diubah saat runtime dengan JavaScript
  • Scoping - Dapat didefinisikan dalam berbagai scope (global atau komponen)
  • Dapat Diwariskan - Nilai diwariskan ke elemen turunan
  • Dapat Dimanipulasi - Mendukung operasi matematika dan fungsi CSS
  • Kompatibilitas - Didukung oleh semua browser modern

1. Dasar-dasar Custom Properties

Custom Properties dideklarasikan dengan awalan -- dan diakses menggunakan fungsi var(). Mereka biasanya didefinisikan dalam selector :root untuk akses global.

CSS 
/* Mendeklarasikan Custom Properties */
:root {
  --primary-color: #3498db;
  --secondary-color: #2ecc71;
  --spacing-unit: 1rem;
  --border-radius: 4px;
  --transition: all 0.3s ease;
}

/* Menggunakan Custom Properties */
.button {
  background-color: var(--primary-color);
  padding: calc(var(--spacing-unit) * 0.5) var(--spacing-unit);
  border-radius: var(--border-radius);
  transition: var(--transition);
}

/* Fallback value */
.element {
  color: var(--text-color, #333); /* Jika --text-color tidak ada, gunakan #333 */
}

2. Scoping dan Inheritance

Custom Properties mengikuti aturan cascade dan inheritance CSS, yang berarti mereka dapat di-scope ke elemen tertentu dan diwariskan ke elemen turunannya.

HTML 
<div class="theme-light">
  <button class="btn">Tombol Light</button>
  
  <div class="theme-dark">
    <button class="btn">Tombol Dark</button>
  </div>
</div>
CSS 
/* Variabel global */
:root {
  --spacing: 1rem;
  --border-radius: 4px;
}

/* Theme Light */
.theme-light {
  --bg-color: #ffffff;
  --text-color: #333333;
  --primary-color: #3498db;
  --border-color: #dddddd;
}

/* Theme Dark */
.theme-dark {
  --bg-color: #1a1a1a;
  --text-color: #f0f0f0;
  --primary-color: #5dade2;
  --border-color: #444444;
}

/* Tombol menggunakan variabel yang di-scope */
.btn {
  padding: calc(var(--spacing) * 0.5) var(--spacing);
  border-radius: var(--border-radius);
  background-color: var(--primary-color);
  color: white;
  border: 1px solid var(--border-color);
  cursor: pointer;
  transition: all 0.3s ease;
}

/* Container untuk preview tema */
.theme-preview {
  background-color: var(--bg-color);
  color: var(--text-color);
  padding: var(--spacing);
  border: 1px solid var(--border-color);
  border-radius: var(--border-radius);
  margin: 1rem 0;
}

3. Manipulasi dengan JavaScript

Salah satu keunggulan utama Custom Properties adalah kemampuannya untuk dimodifikasi secara dinamis menggunakan JavaScript.

JavaScript 
// Mendapatkan nilai Custom Property
const rootStyles = getComputedStyle(document.documentElement);
const primaryColor = rootStyles.getPropertyValue('--primary-color').trim();
console.log('Warna utama:', primaryColor);

// Mengubah nilai Custom Property
document.documentElement.style.setProperty('--primary-color', '#e74c3c');

// Fungsi untuk mengubah tema
function setTheme(theme) {
  const root = document.documentElement;
  
  if (theme === 'dark') {
    root.style.setProperty('--bg-color', '#1a1a1a');
    root.style.setProperty('--text-color', '#f0f0f0');
    root.style.setProperty('--primary-color', '#5dade2');
    root.style.setProperty('--border-color', '#444444');
  } else {
    root.style.setProperty('--bg-color', '#ffffff');
    root.style.setProperty('--text-color', '#333333');
    root.style.setProperty('--primary-color', '#3498db');
    root.style.setProperty('--border-color', '#dddddd');
  }
  
  // Simpan preferensi tema
  localStorage.setItem('theme', theme);
}

// Muat tema yang disimpan
const savedTheme = localStorage.getItem('theme') || 'light';
setTheme(savedTheme);

4. Penggunaan Lanjutan

1. Fallback Values

Gunakan nilai fallback untuk kompatibilitas yang lebih baik: 

.element {
  color: var(--text-color, #333);
  font-size: var(--font-size, 16px);
  margin: var(--spacing, 10px 15px);
}

2. Perhitungan dengan calc()

Custom Properties dapat digunakan dalam fungsi calc(): 

:root {
  --base-spacing: 10px;
  --header-height: 60px;
}

.header {
  height: var(--header-height);
  padding: calc(var(--base-spacing) * 2);
}

.content {
  margin-top: calc(var(--header-height) + var(--base-spacing));
}

3. Manipulasi Warna

Gunakan Custom Properties untuk manipulasi warna: 

:root {
  --primary-hue: 210;
  --primary-saturation: 70%;
  --primary-lightness: 50%;
  
  --primary-color: hsl(
    var(--primary-hue),
    var(--primary-saturation),
    var(--primary-lightness)
  );
  
  --primary-dark: hsl(
    var(--primary-hue),
    var(--primary-saturation),
    calc(var(--primary-lightness) - 10%)
  );
  
  --primary-light: hsl(
    var(--primary-hue),
    var(--primary-saturation),
    calc(var(--primary-lightness) + 10%)
  );
}

.button {
  background-color: var(--primary-color);
  border: 1px solid var(--primary-dark);
}

.button:hover {
  background-color: var(--primary-light);
}
