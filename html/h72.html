Jenis-Jenis Input Formulir HTML

Memahami Berbagai Tipe Input dalam Formulir

HTML menyediakan berbagai tipe input yang dapat digunakan dalam formulir. Setiap tipe input dirancang untuk jenis data tertentu yang membantu dalam validasi dan meningkatkan pengalaman pengguna.

Catatan: Meskipun semua input menggunakan tag <input>, atribut type yang berbeda akan menampilkan kontrol yang berbeda di browser.

1. Input Text (type="text")

Digunakan untuk input teks satu baris seperti nama, alamat, atau teks pendek lainnya.

Contoh Penggunaan:
HTML 
<label for="nama_lengkap">Nama Lengkap:</label>
<input type="text" id="nama_lengkap" name="nama_lengkap" 
       placeholder="Masukkan nama lengkap" maxlength="50">
Atribut Penting:
Atribut Deskripsi Contoh
maxlength Jumlah karakter maksimum yang dapat dimasukkan maxlength="50"
minlength Jumlah karakter minimum yang diperlukan minlength="3"
pattern Ekspresi reguler untuk validasi pattern="[A-Za-z\s]+"
placeholder Teks petunjuk dalam input placeholder="Nama lengkap"
readonly Input hanya bisa dibaca readonly
disabled Menonaktifkan input disabled
size Lebar input dalam karakter size="30"
Tip: Gunakan <textarea> untuk input teks multi-baris seperti komentar atau deskripsi panjang.

2. Input Email (type="email")

Dirancang khusus untuk alamat email dengan validasi otomatis di browser.

Contoh Penggunaan:
Gunakan koma (,) untuk memisahkan beberapa alamat email
HTML 
<label for="email">Alamat Email:</label>
<input type="email" id="email" name="email" 
       placeholder="contoh@email.com" required multiple>
<small>Gunakan koma (,) untuk memisahkan beberapa alamat email</small>
Fitur Khusus:
  • Validasi otomatis format email
  • Keyboard khusus di perangkat mobile
  • Dukungan multiple email dengan atribut multiple
  • Pencocokan domain yang disarankan di beberapa browser
Penting: Selalu lakukan validasi email di sisi server juga, karena validasi HTML dapat di-bypass dengan mudah.

3. Input Password (type="password")

Menyembunyikan karakter yang dimasukkan untuk keamanan.

Contoh Penggunaan:
Minimal 8 karakter
HTML 
<label for="password">Kata Sandi:</label>
<input type="password" id="password" name="password" 
       minlength="8" required>
<small>Minimal 8 karakter</small>

<div style="margin-top: 10px;">
  <label>
    <input type="checkbox" id="showPassword"> Tampilkan kata sandi
  </label>
</div>

<script>
  // Kode JavaScript untuk menampilkan/sembunyikan password
  document.getElementById('showPassword').addEventListener('change', function() {
    const passwordInput = document.getElementById('password');
    passwordInput.type = this.checked ? 'text' : 'password';
  });
</script>
Keamanan Password:
  • Gunakan minimal 8-12 karakter
  • Gabungkan huruf besar, kecil, angka, dan simbol
  • Hindari menggunakan informasi pribadi
  • Gunakan password yang unik untuk setiap akun
Tip: Selalu gunakan HTTPS untuk mengamankan pengiriman data sensitif seperti password melalui internet.

4. Checkbox (type="checkbox")

Memungkinkan pemilihan beberapa opsi dari daftar yang tersedia.

Contoh Penggunaan:

Pilih hobi Anda:

HTML 
<p>Pilih hobi Anda:</p>
<div class="checkbox-group">
  <div class="checkbox-option">
    <input type="checkbox" id="hobi1" name="hobi" value="membaca" checked>
    <label for="hobi1">Membaca</label>
  </div>
  <div class="checkbox-option">
    <input type="checkbox" id="hobi2" name="hobi" value="olahraga">
    <label for="hobi2">Olahraga</label>
  </div>
  <!-- Opsi lainnya -->
</div>
Karakteristik Checkbox:
  • Dapat memilih banyak opsi sekaligus
  • Nilai dikirim ke server hanya jika dicentang
  • Atribut checked membuat opsi terpilih secara default
  • Dapat dikelompokkan dengan fieldset dan legend
Atribut Penting:
  • checked - Opsi terpilih secara default
  • value - Nilai yang akan dikirim ke server
  • required - Minimal satu opsi harus dipilih

5. Radio Buttons (type="radio")

Memungkinkan pemilihan satu opsi dari beberapa pilihan yang tersedia.

Contoh Penggunaan:

Pilih metode pembayaran:

HTML 
<p>Pilih metode pembayaran:</p>
<div class="radio-group">
  <div class="radio-option">
    <input type="radio" id="bayar1" name="pembayaran" value="transfer" checked>
    <label for="bayar1">Transfer Bank</label>
  </div>
  <div class="radio-option">
    <input type="radio" id="bayar2" name="pembayaran" value="kartu">
    <label for="bayar2">Kartu Kredit/Debit</label>
  </div>
  <!-- Opsi lainnya -->
</div>
Karakteristik Radio Button:
  • Hanya satu opsi yang dapat dipilih dalam satu grup
  • Dikelompokkan dengan atribut name yang sama
  • Wajib memiliki atribut value
  • Atribut checked untuk opsi default
Perhatian: Pastikan selalu ada opsi yang terpilih secara default atau gunakan validasi untuk memastikan pengguna memilih setidaknya satu opsi.
Perbandingan Checkbox vs Radio Button:
Fitur Checkbox Radio Button
Jumlah pilihan Bisa memilih banyak Hanya satu yang bisa dipilih
Penggunaan Opsi yang tidak saling eksklusif Opsi yang saling eksklusif
Nilai default Bisa tidak ada yang terpilih Harus ada yang terpilih (disarankan)
Contoh penggunaan Pilihan hobi, filter pencarian Jenis kelamin, metode pembayaran

Praktik Terbaik Penggunaan Input

  • Selalu gunakan <label> untuk setiap input
  • Kelompokkan input terkait dengan <fieldset>
  • Gunakan atribut required untuk bidang wajib
  • Berikan pesan validasi yang jelas
  • Pastikan formulir dapat digunakan dengan keyboard
  • Gunakan tipe input yang sesuai dengan data yang diminta
  • Berikan umpan balik visual saat berinteraksi dengan input
