Mengenal Warna di CSS: color dan background-color

Warna adalah elemen penting dalam desain web yang dapat mempengaruhi tampilan dan nuansa situs web Anda. Di CSS, kita dapat mengatur warna teks menggunakan properti color dan warna latar belakang menggunakan properti background-color.

1. Format Warna di CSS

CSS menyediakan beberapa cara untuk mendefinisikan warna. Mari kita lihat format-format yang tersedia:

a. Nama Warna (Color Keywords)

CSS menyediakan 140+ nama warna yang telah didefinisikan, seperti red, blue, green, tomato, dan rebeccapurple.

tomato
rebeccapurple
dodgerblue
mediumseagreen

b. Kode Heksadesimal

Format heksadesimal menggunakan tanda pagar (#) diikuti oleh 6 digit angka heksadesimal (0-9, A-F). Dua digit pertama mewakili merah, dua berikutnya hijau, dan dua terakhir biru.

CSS 
h1 {
  color: #ff0000;    /* Merah */
}

p {
  color: #00ff00;    /* Hijau */
}

div {
  color: #0000ff;    /* Biru */
}

c. RGB dan RGBA

RGB (Red, Green, Blue) memungkinkan Anda menentukan warna dengan mencampur nilai merah, hijau, dan biru (0-255). RGBA menambahkan saluran alpha untuk transparansi (0-1).

CSS 
/* Warna solid */
.warna-merah {
  background-color: rgb(255, 0, 0);
}

/* Warna dengan transparansi 50% */
.warna-transparan {
  background-color: rgba(0, 0, 255, 0.5);
}

d. HSL dan HSLA

HSL (Hue, Saturation, Lightness) adalah cara lain untuk mendefinisikan warna. HSLA menambahkan saluran alpha.

CSS 
/* Hue: 120 (hijau), Saturation: 100%, Lightness: 50% */
.warna-hijau {
  color: hsl(120, 100%, 50%);
}

/* Dengan transparansi 70% */
.warna-biru-transparan {
  background-color: hsla(240, 100%, 50%, 0.7);
}

2. Properti Warna di CSS

a. color

Properti color mengatur warna teks pada elemen HTML.

CSS 
h1 {
  color: #2c3e50;  /* Warna teks biru tua */
}

.peringatan {
  color: #e74c3c;  /* Warna teks merah */
}

.teks-besar {
  color: rgba(0, 0, 0, 0.8);  /* Hitam dengan transparansi */
}

b. background-color

Properti background-color mengatur warna latar belakang elemen.

CSS 
body {
  background-color: #f8f9fa;  /* Abu-abu muda */
}

.header {
  background-color: #3498db;  /* Biru cerah */
  color: white;
}

.tombol {
  background-color: #2ecc71;  /* Hijau */
  color: white;
  padding: 10px 20px;
  border: none;
  border-radius: 4px;
  cursor: pointer;
}

.tombol:hover {
  background-color: #27ae60;  /* Hijau lebih tua saat hover */
}

3. Gradien Warna

CSS memungkinkan Anda membuat gradien warna yang mulus antara dua warna atau lebih.

a. Linear Gradient

CSS 
.gradient-vertical {
  background: linear-gradient(to bottom, #3498db, #2ecc71);
  height: 100px;
  width: 100%;
}

.gradient-diagonal {
  background: linear-gradient(45deg, #e74c3c, #f39c12, #2ecc71, #3498db);
  height: 100px;
  width: 100%;
  margin: 10px 0;
}

b. Radial Gradient

CSS 
.gradient-radial {
  background: radial-gradient(circle, #3498db, #2c3e50);
  height: 200px;
  width: 200px;
  border-radius: 50%;
  margin: 20px auto;
}

Tips Menggunakan Warna:

  • Gunakan skema warna yang konsisten di seluruh situs web
  • Pastikan kontras yang cukup antara teks dan latar belakang untuk keterbacaan
  • Pertimbangkan aksesibilitas dengan memeriksa rasio kontras
  • Gunakan variabel CSS untuk warna yang sering digunakan
