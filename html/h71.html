Elemen Dasar Formulir HTML

Membuat Formulir Interaktif dengan HTML

Formulir HTML memungkinkan pengguna untuk memasukkan data yang dapat dikirim ke server untuk diproses. Halaman ini akan membahas elemen-elemen dasar yang digunakan dalam pembuatan formulir HTML.

Catatan: Formulir HTML biasanya dikirim ke server menggunakan metode GET atau POST, tetapi untuk contoh ini kita fokus pada elemen-elemen dasarnya terlebih dahulu.

1. Struktur Dasar Formulir

Setiap formulir dimulai dengan tag <form> dan berisi berbagai elemen input di dalamnya.

HTML 
<form action="/proses" method="post">
  <!-- Elemen formulir akan ditempatkan di sini -->
</form>

2. Elemen Input Dasar

HTML 
<label for="nama">Nama Lengkap</label>
<input type="text" id="nama" name="nama" placeholder="Masukkan nama lengkap">
HTML 
<label for="email">Email</label>
<input type="email" id="email" name="email" placeholder="contoh@email.com" required>
HTML 
<label for="password">Kata Sandi</label>
<input type="password" id="password" name="password" minlength="8" required>

3. Textarea

HTML 
<label for="pesan">Pesan</label>
<textarea id="pesan" name="pesan" rows="4" placeholder="Tulis pesan Anda di sini..."></textarea>

</div>

8. Atribut Penting pada Formulir

Atribut Deskripsi Contoh
action URL tempat mengirim data formulir action="/proses.php"
method Metode pengiriman (GET/POST) method="post"
name Nama elemen (untuk pengolahan data) name="nama"
required Wajib diisi required
placeholder Teks petunjuk placeholder="Nama lengkap"
value Nilai default value="contoh@email.com"
disabled Menonaktifkan elemen disabled
readonly Hanya baca (tetap dikirim) readonly

9. Validasi Formulir

HTML5 menyediakan beberapa atribut validasi bawaan:

  • required - Wajib diisi
  • minlength - Panjang teks minimal
  • maxlength - Panjang teks maksimal
  • min/max - Nilai minimal/maksimal untuk angka
  • pattern - Pola regex untuk validasi
  • type - Tipe input (email, number, url, dll)
HTML 
<form id="contohForm">
  <div class="form-group">
    <label for="username">Username (min 4 karakter):</label>
    <input type="text" id="username" name="username" minlength="4" required>
  </div>
  
  <div class="form-group">
    <label for="umur">Umur (13-120):</label>
    <input type="number" id="umur" name="umur" min="13" max="120" required>
  </div>
  
  <div class="form-group">
    <label for="website">Website (opsional):</label>
    <input type="url" id="website" name="website">
  </div>
  
  <div class="form-group">
    <label for="kode_pos">Kode Pos (5 digit):</label>
    <input type="text" id="kode_pos" name="kode_pos" 
           pattern="[0-9]{5}" title="Masukkan 5 digit angka">
  </div>
  
  <button type="submit">Daftar</button>
</form>
Penting: Selalu lakukan validasi juga di sisi server, karena validasi HTML dapat di-bypass dengan mudah.

10. Praktik Terbaik

  • Selalu gunakan tag <label> untuk setiap elemen input
  • Kelompokkan elemen terkait dengan <fieldset> dan beri judul dengan <legend>
  • Gunakan atribut required untuk bidang yang wajib diisi
  • Berikan pesan validasi yang jelas dan membantu
  • Pastikan formulir dapat digunakan dengan keyboard
  • Grupkan elemen terkait secara logis
  • Gunakan tipe input yang sesuai (email, number, dll.)
  • Berikan umpan balik visual saat berinteraksi dengan elemen formulir
