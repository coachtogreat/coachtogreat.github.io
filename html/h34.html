Garis dan Baris Baru

Tag <hr> dan <br> dalam HTML

HTML menyediakan dua tag penting untuk mengatur tata letak teks: <hr> untuk membuat garis horizontal dan <br> untuk membuat baris baru. Mari kita bahas keduanya secara detail.

1. Tag <hr> (Horizontal Rule)

Tag <hr> digunakan untuk membuat garis horizontal yang memisahkan konten. Tag ini bersifat self-closing (tidak memerlukan tag penutup).

Contoh Penggunaan:
HTML 
<h2>Judul Bagian 1</h2>
<p>Ini adalah konten bagian pertama.</p>
<hr>
<h2>Judul Bagian 2</h2>
<p>Ini adalah konten bagian kedua.</p>
Hasil:

Judul Bagian 1

Ini adalah konten bagian pertama.

Judul Bagian 2

Ini adalah konten bagian kedua.

Atribut untuk <hr> (tidak disarankan, gunakan CSS):
  • width - Lebar garis (contoh: width="50%")
  • size - Ketebalan garis (dalam pixel)
  • noshade - Menghilangkan efek bayangan
  • color - Warna garis
  • align - Perataan (left, center, right)
Catatan: Sebaiknya gunakan CSS untuk mengatur tampilan garis horizontal daripada menggunakan atribut HTML langsung.

2. Tag <br> (Line Break)

Tag <br> digunakan untuk membuat baris baru dalam teks. Sama seperti <hr>, tag ini juga bersifat self-closing.

Contoh Penggunaan:
HTML 
<p>
  Baris pertama teks.<br>
  Baris kedua teks dalam paragraf yang sama.<br><br>
  Baris keempat (setelah satu baris kosong).
</p>
Hasil:

Baris pertama teks.
Baris kedua teks dalam paragraf yang sama.

Baris keempat (setelah satu baris kosong).

Kapan Menggunakan <br> dan Kapan Tidak

Gunakan <br> untuk:
  • Alamat
  • Puisi atau lirik lagu
  • Kutipan yang memerlukan jeda baris spesifik
Hindari <br> untuk:
  • Membuat jarak antar paragraf (gunakan margin/padding CSS)
  • Membuat daftar (gunakan <ul>, <ol>, atau <dl>)
  • Layout halaman (gunakan CSS Grid atau Flexbox)

Contoh Praktis

HTML 
<h3>Contoh Alamat</h3>
<address>
  Nama Perusahaan<br>
  Jl. Contoh No. 123<br>
  Kota Contoh, 12345<br>
  Indonesia
</address>

<hr>

<h3>Contoh Puisi</h3>
<p>
  Aku Ingin<br>
  Aku ingin mencintaimu dengan sederhana<br>
  dengan kata yang tak sempat diucapkan<br>
  kayu kepada api yang menjadikannya abu
</p>

Tips Penting

  • Gunakan <hr> untuk memisahkan bagian konten yang memiliki topik berbeda
  • Hindari penggunaan berlebihan <br> untuk tata letak
  • Untuk jarak vertikal yang konsisten, gunakan CSS margin dan padding
  • Gunakan elemen semantik seperti <section> atau <article> untuk memisahkan bagian konten
