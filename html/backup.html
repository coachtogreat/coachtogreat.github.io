Teks dan Paragraf

Tag heading HTML (<h1> sampai <h6>) digunakan untuk mendefinisikan judul dan subjudul pada halaman web. Tag ini menciptakan struktur hierarkis, dengan <h1> sebagai yang paling penting (judul utama) dan <h6> sebagai yang paling tidak penting.

Tag Paragraph (<p>)

Tag <p> adalah elemen blok dasar dalam HTML yang digunakan untuk membuat paragraf teks. Setiap kali Anda menggunakan tag ini, browser akan secara otomatis menambahkan margin di atas dan bawah paragraf, memisahkannya dari elemen-elemen lain di halaman.

Karakteristik Utama:

  • Merupakan block-level element yang menempati satu baris penuh
  • Secara default memiliki margin atas dan bawah sebesar 1em (16px)
  • Dapat berisi teks dan inline elements seperti <strong>, <em>, <a>, dll.
  • Tidak boleh berisi elemen blok lainnya (kecuali di HTML5 yang lebih fleksibel)

Contoh Penggunaan:

<!-- Paragraf sederhana -->
<p>Ini adalah contoh paragraf pertama.</p>

<!-- Paragraf dengan format teks -->
<p>Ini adalah teks <strong>penting</strong> dalam paragraf. 
   Kunjungi <a href="https://example.com">contoh website</a>.</p>

<!-- Paragraf dengan atribut -->
<p class="highlight" style="color: blue;">Paragraf dengan gaya khusus.</p>

Atribut yang Sering Digunakan:

  • align: Mengatur perataan teks (left, right, center, justify) 
    <p align="center">Teks ini rata tengah</p>

    Catatan: Atribut align sudah tidak disarankan di HTML5. Gunakan CSS untuk perataan teks.

  • class dan id: Untuk styling dengan CSS 
    <p class="important" id="paragraf-utama">Paragraf penting</p>
  • title: Memberikan informasi tambahan saat dihover 
    <p title="Informasi tambahan">Arahkan kursor ke sini</p>

Best Practices:

  • Gunakan satu tag <p> untuk setiap paragraf
  • Hindari menggunakan tag <br> untuk membuat spasi antar paragraf
  • Gunakan CSS untuk mengatur tampilan paragraf (margin, padding, dll)
  • Pastikan konten paragraf tetap terbaca dengan baik di berbagai perangkat
