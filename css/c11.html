Pengenalan CSS dan Sejarah Singkat

Apa itu CSS?

CSS (Cascading Style Sheets) adalah bahasa stylesheet yang digunakan untuk mengatur tampilan dan format dokumen HTML. Dengan CSS, Anda dapat mengontrol warna, font, spasi, tata letak, dan banyak aspek lain dari presentasi halaman web.

  • Sekarang: CSS3 terus dikembangkan dengan modul-modul baru

Keuntungan Menggunakan CSS

  • Pemisahan konten dan presentasi: Memisahkan struktur HTML dari gaya
  • Efisiensi: Satu file CSS dapat digunakan untuk banyak halaman
  • Kemudahan perawatan: Perubahan gaya dapat dilakukan di satu tempat
  • Fleksibilitas: Dapat menyesuaikan tampilan untuk berbagai perangkat
  • Kecepatan loading: File CSS di-cache oleh browser

Cara Menambahkan CSS ke HTML

1. Inline CSS

HTML 
<p style="color: blue; font-size: 16px;">Teks berwarna biru dengan ukuran 16px</p>

2. Internal CSS

HTML 
<head>
  <style>
    p {
      color: blue;
      font-size: 16px;
    }
  </style>
</head>

3. Eksternal CSS (Direkomendasikan)

HTML 
<head>
  <link rel="stylesheet" href="styles.css">
</head>
CSS (styles.css) 
p {
  color: blue;
  font-size: 16px;
}

Syntax Dasar CSS

CSS 
selector {
  property: value;
  /* Komentar CSS */
}
  • Selector: Elemen HTML yang ingin di-styling
  • Property: Aspek yang ingin diubah (warna, ukuran, dll)
  • Value: Nilai dari property yang dipilih

Contoh CSS Sederhana

CSS 
body {
  font-family: Arial, sans-serif;
  line-height: 1.6;
  color: #333;
  max-width: 800px;
  margin: 0 auto;
  padding: 20px;
}

h1 {
  color: #2c3e50;
  border-bottom: 2px solid #3498db;
  padding-bottom: 10px;
}

.tombol {
  background-color: #3498db;
  color: white;
  padding: 10px 20px;
  border: none;
  border-radius: 4px;
  cursor: pointer;
}

.tombol:hover {
  background-color: #2980b9;
}

Dengan memahami dasar-dasar CSS ini, Anda sudah siap untuk mulai mempercantik halaman web Anda. Di bab selanjutnya, kita akan mempelajari lebih dalam tentang selector, properti, dan nilai-nilai CSS.
