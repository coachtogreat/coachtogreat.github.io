Unit Responsif di CSS: %, em, rem, vh, vw

Unit responsif di CSS memungkinkan kita membuat tata letak yang fleksibel dan adaptif terhadap berbagai ukuran layar. Berbeda dengan unit tetap seperti px, unit responsif menyesuaikan ukurannya berdasarkan konteks tertentu.

Jenis-Jenis Unit Responsif:

  • % - Persentase dari elemen induk
  • em - Relatif terhadap ukuran font elemen saat ini
  • rem - Relatif terhadap ukuran font elemen root (html)
  • vh - 1% dari tinggi viewport
  • vw - 1% dari lebar viewport

1. Persentase (%)

Unit persentase relatif terhadap ukuran elemen induknya. Berguna untuk membuat lebar yang fleksibel.

CSS 
.container {
  width: 80%; /* 80% dari lebar induk */
  margin: 0 auto;
}

.box {
  width: 50%; /* 50% dari lebar .container */
  padding: 2%; /* 2% dari lebar elemen */
}

2. Unit em

em relatif terhadap ukuran font elemen saat ini. Jika digunakan untuk properti selain font-size, 1em sama dengan ukuran font yang dihitung dari elemen saat ini.

CSS 
body {
  font-size: 16px;
}

.box {
  font-size: 1.5em; /* 24px (16 * 1.5) */
  margin: 1em; /* 24px (sama dengan font-size elemen ini) */
  padding: 0.5em; /* 12px */
}

/* Contoh efek kompounding */
.nested-box {
  font-size: 2em; /* 48px (24 * 2) */
}

Perhatian: Penggunaan em yang bersarang dapat menyebabkan efek kompounding di mana ukuran menjadi semakin besar/kecil secara eksponensial.

3. Unit rem (Root em)

rem selalu relatif terhadap ukuran font elemen root (html). Menghindari masalah kompounding yang terjadi pada em.

CSS 
html {
  font-size: 16px; /* 1rem = 16px */
}

.box {
  font-size: 1.5rem; /* 24px */
  padding: 1rem; /* 16px */
  margin: 0.5rem; /* 8px */
}

/* Tidak ada efek kompounding */
.nested-box {
  font-size: 2rem; /* 32px */
}

em vs rem

em: Relatif terhadap font-size elemen induk terdekat yang memiliki font-size ditentukan.

rem: Selalu relatif terhadap font-size elemen root (html).

Kapan Menggunakan?

Gunakan em: Untuk properti yang ingin mengikuti font-size elemen.

Gunakan rem: Untuk ukuran yang konsisten di seluruh halaman, terutama untuk spacing dan layout.

4. Viewport Units: vh dan vw

Viewport units relatif terhadap ukuran viewport (area yang terlihat di browser).

CSS 
.header {
  height: 100vh; /* 100% tinggi viewport */
  width: 100vw; /* 100% lebar viewport */
}

.section {
  min-height: 50vh; /* Minimal 50% tinggi viewport */
  padding: 5vw; /* Padding 5% dari lebar viewport */
}

.box {
  width: 50vw; /* 50% lebar viewport */
  height: 25vh; /* 25% tinggi viewport */
}

5. Unit Responsif Lainnya

Unit Tambahan:

  • vmin: 1% dari ukuran viewport yang lebih kecil (lebar atau tinggi)
  • vmax: 1% dari ukuran viewport yang lebih besar (lebar atau tinggi)
  • ch: Lebar karakter "0" (nol) dari font saat ini
  • lh: Tinggi baris (line-height) elemen saat ini

Best Practices:

  • Gunakan rem untuk ukuran font dan spacing untuk konsistensi
  • Gunakan em untuk properti yang perlu diskalakan dengan ukuran font
  • Gunakan % untuk lebar dan tinggi yang fleksibel dalam container
  • Gunakan vh/vw untuk elemen yang perlu mengisi viewport
  • Hindari mencampur terlalu banyak jenis unit dalam satu komponen
