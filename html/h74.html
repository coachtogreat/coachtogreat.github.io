Tombol Submit & Reset

Mengontrol Pengiriman dan Pengaturan Ulang Formulir

Halaman ini akan membahas secara mendalam tentang tombol submit dan reset dalam formulir HTML, termasuk berbagai cara menggunakannya, memodifikasi perilakunya, dan praktik terbaik dalam pengembangan web modern.

1. Tombol Submit (<input type="submit"> atau <button type="submit">)

Tombol untuk mengirim data formulir ke server.

Contoh Dasar Tombol Submit:
HTML 
<!-- Menggunakan input -->
<input type="submit" value="Kirim Formulir">

<!-- Atau menggunakan button -->
<button type="submit">Kirim dengan Button</button>
Atribut Penting Tombol Submit:
Atribut Deskripsi Contoh
type Harus diisi dengan "submit" type="submit"
value Teks yang ditampilkan pada tombol value="Kirim"
form ID formulir yang terkait (jika di luar form) form="form1"
formaction Mengganti URL pengiriman form formaction="/proses"
formmethod Metode HTTP (get/post/put/delete) formmethod="post"
formtarget Dimana menampilkan respon formtarget="_blank"
disabled Menonaktifkan tombol disabled
Contoh Lanjutan Tombol Submit:
HTML 
<!-- Tombol dengan ikon -->
<button type="submit" class="btn-icon">
  <span>Kirim Sekarang</span>
  <span>‚Üí</span>
</button>

<!-- Tombol outline -->
<button type="submit" class="btn-outline-primary">
  Simpan Sebagai Draft
</button>

<!-- Tombol lebar penuh -->
<button type="submit" class="btn-block">
  Daftar Sekarang
</button>

2. Tombol Reset (<input type="reset"> atau <button type="reset">)

Tombol untuk mengembalikan nilai form ke nilai default.

Contoh Dasar Tombol Reset:
HTML 
<form id="form3">
  <div class="form-group">
    <label for="kota">Kota Asal:</label>
    <input type="text" id="kota" name="kota" value="Jakarta">
  </div>
  <div class="form-group">
    <label for="negara">Negara:</label>
    <select id="negara" name="negara">
      <option value="id">Indonesia</option>
      <option value="my">Malaysia</option>
      <option value="sg">Singapura</option>
    </select>
  </div>
  <div class="btn-group">
    <input type="submit" value="Kirim">
    <input type="reset" value="Reset Form">
  </div>
</form>
Perhatian: Tombol reset akan mengembalikan semua nilai form ke nilai default, yang bisa membuat pengguna kehilangan data yang sudah diisi. Gunakan dengan hati-hati dan pertimbangkan untuk menggunakan konfirmasi JavaScript sebelum mengosongkan form.
Contoh dengan Konfirmasi Reset:
HTML & JavaScript 
<form id="form4">
  <div class="form-group">
    <label for="komentar">Komentar:</label>
    <textarea id="komentar" name="komentar" rows="3">Coba ubah teks ini lalu klik Reset</textarea>
  </div>
  <div class="btn-group">
    <button type="submit">Kirim Komentar</button>
    <button type="button" onclick="if(confirm('Yakin ingin mengosongkan form?')) this.form.reset();">
      Reset dengan Konfirmasi
    </button>
  </div>
</form>

4. Validasi Form Sebelum Submit

Menggunakan JavaScript untuk memvalidasi form sebelum dikirim.

Contoh Validasi Form:
Minimal 3 karakter
Minimal 6 karakter
HTML & JavaScript 
<form id="validasiForm" onsubmit="return validasiForm()">
  <div class="form-group">
    <label for="username">Username:</label>
    <input type="text" id="username" name="username" required minlength="3">
    <small>Minimal 3 karakter</small>
  </div>
  <div class="form-group">
    <label for="password">Password:</label>
    <input type="password" id="password" name="password" required minlength="6">
    <small>Minimal 6 karakter</small>
  </div>
  <div class="form-group">
    <label for="konfirmasi">Konfirmasi Password:</label>
    <input type="password" id="konfirmasi" name="konfirmasi" required>
  </div>
  <div class="btn-group">
    <button type="submit">Daftar</button>
    <button type="reset">Reset</button>
  </div>
</form>

<script>
function validasiForm() {
  const username = document.getElementById('username').value;
  const password = document.getElementById('password').value;
  const konfirmasi = document.getElementById('konfirmasi').value;
  
  if (username.length < 3) {
    alert('Username minimal 3 karakter');
    return false;
  }
  
  if (password.length < 6) {
    alert('Password minimal 6 karakter');
    return false;
  }
  
  if (password !== konfirmasi) {
    alert('Konfirmasi password tidak cocok');
    return false;
  }
  
  alert('Pendaftaran berhasil!');
  return false; // Hanya untuk contoh, hapus return false di produksi
}
</script>

Praktik Terbaik Penggunaan Tombol Formulir

  • Gunakan teks yang jelas dan deskriptif pada tombol
  • Hindari menggunakan "Klik di sini" - jelaskan apa yang akan terjadi
  • Buat tombol submit lebih menonjol dari tombol lainnya
  • Berikan umpan balik visual saat tombol ditekan
  • Nonaktifkan tombol submit setelah diklik untuk mencegah pengiriman ganda
  • Gunakan ikon yang sesuai untuk meningkatkan kejelasan
  • Pastikan tombol cukup besar untuk mudah ditekan di perangkat layar sentuh
  • Berikan cukup ruang di sekitar tombol untuk menghindari kesalahan ketuk
  • Gunakan warna yang sesuai dengan maknanya (hijau untuk konfirmasi, merah untuk peringatan, dll)
  • Uji tombol di berbagai perangkat dan ukuran layar
