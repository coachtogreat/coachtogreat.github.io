Daftar Tidak Berurutan (Unordered List) di HTML & CSS

Daftar tidak berurutan (Unordered List) adalah elemen HTML yang digunakan untuk menampilkan daftar item tanpa urutan tertentu. Setiap item dalam daftar biasanya diawali dengan bullet atau simbol lainnya.

Konsep Dasar Unordered List:

  • <ul>: Tag pembungkus untuk membuat daftar tidak berurutan (unordered list)
  • <li>: Tag untuk setiap item dalam daftar (list item)
  • Atribut type: Menentukan jenis bullet (disc, circle, square, none)
  • list-style-type: Properti CSS untuk mengubah tampilan bullet
  • list-style-image: Properti CSS untuk menggunakan gambar sebagai bullet

1. Daftar Tidak Berurutan Dasar

Berikut adalah contoh sederhana penggunaan unordered list dengan bullet default:

HTML 
<ul>
  <li>Nasi Goreng</li>
  <li>Mie Goreng</li>
  <li>Soto Ayam</li>
  <li>Gado-gado</li>
</ul>

Hasil:

  • Nasi Goreng
  • Mie Goreng
  • Soto Ayam
  • Gado-gado

2. Jenis Bullet

Anda dapat mengubah jenis bullet menggunakan properti CSS list-style-type:

list-style-type:

Custom Bullet:

CSS 
ul.custom-list {
  list-style-type: disc;
  padding-left: 20px;
}

Hasil:

  • Item pertama
  • Item kedua
  • Item ketiga

3. Styling dengan CSS

Anda dapat mengubah tampilan daftar tidak berurutan menggunakan CSS:

CSS 
/* Menggunakan karakter khusus sebagai bullet */
.checkmark-list {
  list-style: none;
  padding-left: 0;
}

.checkmark-list li {
  position: relative;
  padding-left: 30px;
  margin-bottom: 10px;
}

.checkmark-list li::before {
  content: '✓';
  position: absolute;
  left: 0;
  color: #4caf50;
  font-weight: bold;
}

/* Menggunakan gambar sebagai bullet */
.custom-bullet {
  list-style-image: url('path/ke/gambar-bullet.png');
  padding-left: 25px;
}

/* Menghilangkan bullet dan menambahkan border */
.no-bullet {
  list-style: none;
  padding-left: 0;
}

.no-bullet li {
  padding: 8px 0;
  border-bottom: 1px solid #e0e0e0;
}

/* Horizontal list */
.horizontal-list {
  list-style: none;
  padding: 0;
  display: flex;
  gap: 15px;
}

.horizontal-list li {
  background-color: #f0f0f0;
  padding: 5px 10px;
  border-radius: 4px;
}

Contoh Penggunaan:

Daftar dengan Checkmark:
  • Fitur pertama
  • Fitur kedua
  • Fitur ketiga
Daftar Horizontal:
  • Beranda
  • Tentang
  • Layanan
  • Kontak
Daftar Tanpa Bullet dengan Border:
  • Item 1
  • Item 2
  • Item 3

4. Daftar Bersarang (Nested List)

Anda dapat membuat daftar di dalam daftar untuk membuat hierarki yang lebih kompleks:

HTML 
<ul>
  <li>Perangkat Elektronik
    <ul>
      <li>Smartphone
        <ul>
          <li>Android</li>
          <li>iOS</li>
        </ul>
      </li>
      <li>Laptop
        <ul>
          <li>Windows</li>
          <li>MacOS</li>
          <li>Linux</li>
        </ul>
      </li>
    </ul>
  </li>
  <li>Perabotan Rumah
    <ul>
      <li>Kursi</li>
      <li>Meja</li>
      <li>Lemari</li>
    </ul>
  </li>
</ul>

Hasil:

  • Perangkat Elektronik
    • Smartphone
      • Android
      • iOS
    • Laptop
      • Windows
      • MacOS
      • Linux
  • Perabotan Rumah
    • Kursi
    • Meja
    • Lemari
