Console Debugging

1. Pengenalan Console

Cara Mengakses Console 
// 1. Mengakses Console
// - Tekan F12 atau Ctrl+Shift+I
// - Pilih tab "Console"
// - Atau klik kanan di halaman web dan pilih "Inspect"

// 2. Menulis ke Console
console.log('Halo, ini pesan log');
console.error('Ini pesan error');
console.warn('Ini pesan warning');
console.info('Ini pesan info');

2. Metode Console

Metode Utama Console 
// 1. console.log()
console.log('Pesan', 'dengan', 'beberapa', 'argumen');
console.log({ nama: 'John', umur: 30 });

// 2. console.error()
console.error('Terjadi kesalahan');

// 3. console.warn()
console.warn('Peringatan!');

// 4. console.info()
console.info('Informasi penting');

// 5. console.debug()
console.debug('Pesan debug');

// 6. console.table()
const data = [
    { nama: 'John', umur: 30 },
    { nama: 'Jane', umur: 25 }
];
console.table(data);

3. Debugging dengan Breakpoints

Penggunaan Breakpoints 
// 1. Menggunakan debugger;
function hitung(a, b) {
    debugger;
    return a + b;
}

// 2. Menggunakan console.trace()
function fungsi1() {
    fungsi2();
}

function fungsi2() {
    console.trace();
}

// 3. Menggunakan console.time() dan console.timeEnd()
console.time('Waktu Eksekusi');
for (let i = 0; i < 1000000; i++) {
    // Operasi
}
console.timeEnd('Waktu Eksekusi');

// 4. Menggunakan console.assert()
const nilai = 10;
console.assert(nilai > 5, 'Nilai harus lebih besar dari 5');

4. Tips Debugging Efektif

  • Gunakan console.log() dengan bijak
  • Gunakan debugger; untuk men-debug kode
  • Gunakan console.trace() untuk melihat stack trace
  • Gunakan console.time() untuk mengukur performa
  • Gunakan console.assert() untuk validasi
  • Gunakan console.table() untuk menampilkan data tabel
  • Gunakan console.group() untuk mengelompokkan log
  • Gunakan console.error() untuk pesan error

5. Contoh Praktis

Contoh Debugging Aplikasi 
// 1. Debugging API Call
async function fetchData() {
    console.time('API Call');
    try {
        const response = await fetch('https://api.example.com/data');
        console.log('Response status:', response.status);
        const data = await response.json();
        console.log('Data received:', data);
        return data;
    } catch (error) {
        console.error('Error:', error);
        throw error;
    } finally {
        console.timeEnd('API Call');
    }
}

// 2. Debugging Loop
function prosesData(data) {
    console.group('Proses Data');
    console.log('Data masuk:', data);
    
    for (let i = 0; i < data.length; i++) {
        console.log(`Mengolah item ${i}:`, data[i]);
        // Operasi
    }
    
    console.groupEnd();
}

// 3. Debugging Event
function handleClick(event) {
    console.log('Event:', event);
    console.log('Target:', event.target);
    console.log('Type:', event.type);
}

6. Best Practices

  • Gunakan console.group() untuk mengorganisir log
  • Tambahkan timestamp ke log penting
  • Gunakan console.trace() untuk melacak stack
  • Gunakan console.time() untuk mengukur performa
  • Gunakan console.assert() untuk validasi
  • Hapus console.log() sebelum deployment
  • Gunakan console.error() untuk error handling
  • Gunakan console.warn() untuk peringatan
