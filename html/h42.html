Atribut Tautan HTML

Memahami Atribut <a> dalam HTML

Tag <a> memiliki berbagai atribut yang memungkinkan Anda mengontrol perilaku tautan. Mari kita bahas setiap atribut penting beserta penggunaannya.

1. Atribut href (Hypertext Reference)

Atribut paling penting yang menentukan tujuan tautan.

HTML 
<!-- Tautan ke halaman web -->
<a href="https://contoh.com">Kunjungi Contoh.com</a>

<!-- Tautan ke bagian dalam halaman -->
<a href="#bagian2">Ke Bagian 2</a>

<!-- Tautan email -->
<a href="mailto:email@contoh.com">Kirim Email</a>

<!-- Tautan telepon -->
<a href="tel:+62123456789">Telepon Kami</a>

<!-- Tautan untuk mengunduh file -->
<a href="dokumen.pdf" download>Unduh PDF</a>

2. Atribut target

Menentukan di mana dokumen yang ditautkan akan dibuka.

HTML 
<!-- Buka di tab/jendela baru -->
<a href="https://contoh.com" target="_blank">Buka di Tab Baru</a>

<!-- Buka di frame induk -->
<a href="halaman.html" target="_parent">Buka di Frame Induk</a>

<!-- Buka di jendela saat ini (default) -->
<a href="halaman.html" target="_self">Buka di Jendela Ini</a>

<!-- Buka di frame tertentu -->
<a href="halaman.html" target="nama-frame">Buka di Frame Tertentu</a>

3. Atribut rel (Relationship)

Menentukan hubungan antara dokumen saat ini dan dokumen yang ditautkan.

HTML 
<!-- Mencegah serangan tabnabbing -->
<a href="https://eksternal.com" target="_blank" rel="noopener noreferrer">
  Tautan Eksternal Aman
</a>

<!-- Tautan nofollow untuk SEO -->
<a href="https://iklan.com" rel="nofollow">Iklan</a>

<!-- Tautan sponsor -->
<a href="https://sponsor.contoh" rel="sponsored">Sponsor</a>

<!-- Tautan UGC (User Generated Content) -->
<a href="#komentar" rel="ugc">Lihat Komentar</a>

4. Atribut download

Memaksa browser untuk mengunduh file alih-alih menampilkannya.

HTML 
<!-- Unduh dengan nama file default -->
<a href="laporan.pdf" download>Unduh Laporan</a>

<!-- Tentukan nama file unduhan -->
<a href="laporan-januari-2023.pdf" download="laporan-terbaru.pdf">
  Unduh Laporan Terbaru
</a>

5. Atribut title

Menampilkan tooltip saat pengguna mengarahkan kursor ke tautan.

HTML 
<a href="#" title="Ini adalah tooltip">Arahkan kursor ke sini</a>

6. Atribut hreflang

Menentukan bahasa dokumen yang ditautkan.

HTML 
<!-- Tautan ke versi bahasa Inggris -->
<a href="https://example.com/en" hreflang="en">English Version</a>

<!-- Tautan ke versi bahasa Indonesia -->
<a href="https://example.com/id" hreflang="id">Versi Indonesia</a>

7. Atribut ping

Mengirim permintaan POST kecil ke URL yang ditentukan saat tautan diklik.

HTML 
<a href="https://contoh.com" ping="/track">
  Tautan dengan Pelacakan
</a>

8. Atribut referrerpolicy

Mengontrol berapa banyak informasi referrer yang dikirim dengan tautan.

HTML 
<!-- Tidak mengirim informasi referrer -->
<a href="https://contoh.com" referrerpolicy="no-referrer">
  Tanpa Referrer
</a>

<!-- Hanya mengirim origin -->
<a href="https://contoh.com" referrerpolicy="origin">
  Hanya Origin
</a>

Praktik Terbaik

  • Selalu sertakan rel="noopener noreferrer" saat menggunakan target="_blank"
  • Gunakan download dengan hati-hati, pastikan pengguna mengharapkan file diunduh
  • Gunakan title untuk aksesibilitas, bukan sebagai pengganti teks tautan yang jelas
  • Hindari membuka tautan di jendela baru kecuali benar-benar diperlukan
  • Gunakan hreflang untuk situs multibahasa
