Struktur Dasar HTML

Komentar dalam HTML?

Komentar HTML adalah bagian dari kode HTML yang tidak akan ditampilkan di browser. Komentar digunakan untuk memberikan catatan, penjelasan, atau petunjuk dalam kode, baik untuk diri sendiri maupun untuk developer lain.

Sintaks Komentar dalam HTML

HTML 
<!-- Ini adalah komentar -->

Ciri-ciri:

  • Dimmulai dengan <!--
  • Diakhir dengan -->
  • Semua teks diantaranya tidak akan diproses atau ditampilkan di browser

Contoh Penggunaan Komentar

Contoh sederhana:

HTML 
<!-- Ini adalah bagian header -->
            <header>
              <h1>Selamat Datang</h1>
            </header>

Menonaktifkan HTML

HTML 
<!-- 
Paragraf ini tidak akan tampil di browser
 -->

Kapan komentar digunakan?

Kebutuhan Contoh
Menjelaskan bagian kode <!-- Navigasi utama website -->
Memberi catatan TODO <!-- TODO: Tambahkan login form -->
Menonaktifkan sementara kode <!-- <script src="app.js"></script> -->
Menjelaskan struktur kompleks <!-- Grid layout dimulai di sini -->
Kolaborasi (komentar internal) <!-- Dikerjakan oleh tim frontend pada 2025 -->

Hal yang perlu dihindari

⚠️ Jangan gunakan -- di dalam komentar.

HTML 
<!-- Salah: Jangan lakukan ini -- karena bisa error -->

HTML tidak mengizinkan penggunaan -- di dalam isi komentar.


⚠️ Jangan tempatkan komentar di dalam tag HTML:

HTML 
<!-- Salah -->
            <p <!-- komentar -->  >Teks</p>

Komentar tidak boleh memecah struktur tag.

Tips Profesional:

✅ Gunakan komentar untuk:

  • Strukutr kode kompleks
  • Dokumentasi internal
  • Menjaga kolaborasi tim tetap jelas

❌ Jangan gunakan komentar untuk:

  • Menyembunyikan data sensiti
  • Dokumentasi panjang yang lebih cocok di file README
