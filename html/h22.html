Struktur Dasar HTML

Apa itu Atribut HTML?

Atribut HTML adalah komponen penting yang memberikan informasi tambahan tentang elemen HTML. Setiap atribut memiliki fungsi spesifik yang memodifikasi perilaku atau tampilan elemen tersebut. Atribut selalu didefinisikan dalam tag pembuka elemen HTML dan biasanya terdiri dari pasangan nama dan nilai (name="value").

Penggunaan atribut yang tepat sangat penting untuk:

  • Meningkatkan aksesibilitas web
  • Memperbaiki SEO (Search Engine Optimization)
  • Mengontrol perilaku dan tampilan elemen
  • Menambahkan metadata dan informasi tambahan
HTML 
<tagname atribut="nilai">Konten</tagname>

Struktur Dasar Atribut HTML

HTML 
<elemen nama_atribut="nilai_atribut">Isi</elemen>

Komponen Utama Atribut HTML:

  • Nama Atribut: Mengidentifikasi jenis atribut (contoh: href, src, alt, class, id). Nama atribut bersifat case-insensitive tetapi sebaiknya menggunakan huruf kecil.
  • Tanda Sama Dengan (=): Memisahkan nama atribut dengan nilainya.
  • Nilai Atribut:
    • Biasanya diapit tanda kutip ganda ("), bisa juga kutip tunggal (').
    • Untuk nilai yang berisi spasi, harus menggunakan tanda kutip.
    • Beberapa atribut tidak memerlukan nilai (boolean attributes).

Jenis-jenis Atribut HTML

Atribut HTML dapat dikelompokkan menjadi beberapa kategori berdasarkan fungsinya:

1. Atribut Dasar

Atribut yang umum digunakan di hampir semua elemen HTML:

Atribut Elemen yang Umum Digunakan Deskripsi
id Semua elemen Memberikan identitas unik ke elemen.
class Semua elemen Menentukan satu atau lebih kelas CSS.
style Semua elemen Menambahkan gaya CSS langsung pada elemen (inline styling).
title Semua elemen Memberikan informasi tambahan saat hover.
href <a> Menentukan tujuan tautan (URL).
src <img>, <script>, dll. Menentukan sumber eksternal (gambar, script, dll).
alt <img> Teks alternatif saat gambar tidak bisa ditampilkan.
type <input>, <button> Menentukan tipe input (seperti text, password, submit, dll).
value <input>, <option> Menentukan nilai awal elemen.
name <input>, <form> Nama elemen, penting saat mengirim data form.
placeholder <input>, <textarea> Teks petunjuk dalam input sebelum user mengetik.
disabled Form elements Menonaktifkan input/form.
readonly <input>, <textarea> Hanya bisa dibaca, tidak bisa diubah.
checked <input type="checkbox"> Menentukan apakah checkbox tercentang.
selected <option> Menentukan opsi yang dipilih secara default.
required Form elements Wajib diisi sebelum form bisa dikirim.
action <form> URL tujuan saat form dikirim.
method <form> Metode pengiriman (GET atau POST).
target <a>, <form> Menentukan di mana membuka tautan/form (misalnya: _blank, _self).

2. Atribut Global

Atribut yang dapat digunakan pada semua elemen HTML, terlepas dari jenis elemennya. Atribut ini mendukung aksesibilitas, penanganan kejadian, dan fungsionalitas dasar lainnya.

Atribut Deskripsi
id Identifikasi unik.
class Kelas CSS.
style Gaya CSS langsung.
title Tooltip saat hover.
hidden Menyembunyikan elemen dari tampilan.
data-* Atribut khusus untuk menyimpan data kustom (digunakan di JS).
tabindex Urutan tab navigasi.
contenteditable Membuat elemen bisa diedit.
draggable Menentukan apakah elemen bisa di-drag.
lang Menentukan bahasa konten (en, id, dsb).
dir Arah teks: ltr (kiri ke kanan), rtl (kanan ke kiri).
accesskey Shortcut keyboard untuk elemen.
role Peran aksesibilitas elemen (digunakan untuk screen reader).

3. Atribut Boolean

Atribut boolean adalah atribut yang tidak memerlukan nilai. Keberadaan atribut ini saja sudah cukup untuk mengaktifkan fungsinya.

  • disabled - Menonaktifkan elemen form
  • readonly - Membuat input hanya bisa dibaca
  • required - Menjadikan input wajib diisi
  • autofocus - Fokus otomatis ke elemen saat halaman dimuat
  • multiple - Memungkinkan banyak nilai untuk input

4. Atribut Khusus Elemen

Atribut yang hanya berlaku untuk elemen tertentu dan memiliki fungsi spesifik untuk elemen tersebut.

  • href, target - Khusus elemen <a> (link)
  • src, alt - Khusus elemen <img> (gambar)
  • action, method - Khusus elemen <form>
  • colspan, rowspan - Khusus elemen tabel

Contoh Penggunaan Praktis

HTML 
<!-- Link dengan title dan target -->
<a href="https://contoh.com" target="_blank" title="Kunjungi Situs Contoh">
  Kunjungi Situs Kami
</a>

<!-- Input teks dengan placeholder dan required -->
<input 
  type="email" 
  name="email" 
  placeholder="Masukkan email Anda" 
  required
>

<!-- Gambar dengan alt text dan dimensi -->
<img 
  src="foto-profil.jpg" 
  alt="Foto Profil Pengguna" 
  width="250" 
  height="250"
  class="profile-pic"
>

Best Practices Penggunaan Atribut

  • Atribut Boolean: Tidak perlu nilai, cukup tulis nama atributnya saja. 
    <input type="checkbox" checked>
<button disabled>Submit</button>
  • Atribut Data Kustom: Gunakan awalan data- untuk menyimpan data khusus. 
    <div data-user-id="1234" data-role="admin"></div>
  • Atribut ARIA: Gunakan untuk meningkatkan aksesibilitas. 
    <button aria-label="Tutup menu">Ã—</button>
  • Atribut Performa: Seperti loading="lazy" untuk gambar dan iframe. 
    <img src="gambar-besar.jpg" loading="lazy" alt="Deskripsi">
