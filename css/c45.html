Memahami Grid Layout di CSS

CSS Grid Layout adalah sistem tata letak dua dimensi yang memungkinkan Anda membuat desain web yang kompleks dan responsif dengan mudah. Berbeda dengan Flexbox yang satu dimensi, Grid memungkinkan pengaturan baik baris maupun kolom secara bersamaan.

Konsep Dasar Grid:

  • Grid Container: Elemen induk yang menampung item-item grid
  • Grid Item: Elemen-elemen anak langsung dari grid container
  • Grid Line: Garis pembatas yang membentuk struktur grid
  • Grid Track: Ruang antara dua garis grid yang berdekatan (baris atau kolom)
  • Grid Cell: Perpotongan antara satu baris dan satu kolom
  • Grid Area: Satu atau lebih sel grid yang membentuk area persegi panjang

1. Membuat Grid Container

Untuk memulai menggunakan Grid, Anda perlu mengubah elemen menjadi grid container dengan menetapkan display: grid atau display: inline-grid.

CSS 
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  grid-template-rows: 100px 200px;
  gap: 10px;
  padding: 10px;
}

.item {
  background-color: #2196f3;
  color: white;
  padding: 20px;
  text-align: center;
}

2. Penempatan Item di Grid

Anda dapat mengontrol posisi dan ukuran item grid menggunakan properti grid-column dan grid-row.

CSS 
.header {
  grid-column: 1 / 4;  /* Mulai dari line 1 sampai 4 */
  grid-row: 1;
}

.sidebar {
  grid-column: 1;
  grid-row: 2 / 4;  /* Mulai dari line 2 sampai 4 */
}

.main {
  grid-column: 2 / 4;
  grid-row: 2;
}

.footer {
  grid-column: 1 / 4;
  grid-row: 3;
}

3. Area Grid

Grid Template Areas memungkinkan Anda mendefinisikan layout menggunakan nama-nama area yang deskriptif.

CSS 
.container {
  display: grid;
  grid-template-areas:
    "header header header"
    "sidebar main main"
    "footer footer footer";
  grid-template-columns: 200px 1fr 1fr;
  grid-template-rows: 80px 1fr 80px;
  gap: 10px;
  height: 400px;
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.footer { grid-area: footer; }

4. Grid Responsif

Grid dapat dengan mudah dibuat responsif menggunakan media queries dan fungsi repeat() dengan auto-fill/fit.

CSS 
.grid-container {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
  gap: 20px;
  padding: 20px;
}

/* Untuk layar lebih kecil dari 600px */
@media (max-width: 600px) {
  .grid-container {
    grid-template-columns: 1fr;
  }
}
