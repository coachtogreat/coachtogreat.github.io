Variabel dalam JavaScript

1. Pengenalan Variabel

Variabel adalah wadah untuk menyimpan data dalam program JavaScript. JavaScript memiliki tiga cara untuk mendeklarasikan variabel: var, let, dan const.

2. Perbedaan var, let, dan const

Kata Kunci Scope Reassignment Hoisting Temporal Dead Zone
var Function Scope Boleh Ya Tidak
let Block Scope Boleh Ya Ya
const Block Scope Tidak bisa Ya Ya

3. Contoh Penggunaan var

Contoh var 
// Deklarasi dengan var
var nama = "John";

// Bisa diubah
nama = "Jane";

// Scope function
function test() {
    var x = 5;
    if (true) {
        var x = 10; // masih bisa diakses di luar if
    }
    console.log(x); // output: 10
}

4. Contoh Penggunaan let

Contoh let 
// Deklarasi dengan let
let nama = "John";

// Bisa diubah
nama = "Jane";

// Scope block
function test() {
    let x = 5;
    if (true) {
        let x = 10; // hanya bisa diakses di dalam if
    }
    console.log(x); // output: 5
}

5. Contoh Penggunaan const

Contoh const 
// Deklarasi dengan const
const PI = 3.14159;
// PI = 3.14; // Error: tidak bisa diubah

// Objek dengan const
const orang = {
    nama: "John",
    umur: 25
};

// Bisa mengubah properti
orang.nama = "Jane"; // OK

// Tidak bisa mengubah referensi
orang = { nama: "Mike" }; // Error: tidak bisa diubah

6. Best Practices

  • Gunakan const sebagai default
  • Gunakan let jika nilai perlu diubah
  • Hindari penggunaan var kecuali diperlukan
  • Deklarasikan variabel sebelum digunakan
  • Beri nama variabel yang deskriptif

7. Contoh Praktis

Contoh Praktis 
// Contoh penggunaan variabel dalam fungsi
function hitungLuasPersegi(sisi) {
    const PI = 3.14159;
    let luas = sisi * sisi;
    return luas;
}

// Contoh penggunaan variabel dalam loop
const angka = [1, 2, 3, 4, 5];
for (let i = 0; i < angka.length; i++) {
    console.log(angka[i]);
}

8. Kesalahan Umum

  • Menggunakan variabel tanpa deklarasi
  • Menggunakan nama variabel yang sama di scope yang sama
  • Mengubah nilai dari const
  • Menggunakan var di ES6+ tanpa alasan khusus
  • Tidak memberi nilai saat deklarasi const
