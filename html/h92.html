Memasukkan Audio di HTML

Panduan Lengkap Menambahkan Audio ke Halaman Web

HTML5 menyediakan elemen <audio> yang memudahkan kita untuk menambahkan file audio ke dalam halaman web. Mari kita pelajari cara menggunakannya dengan benar.

Catatan: Format audio yang didukung bervariasi tergantung browser. Format yang umum didukung adalah MP3, WAV, dan OGG.

1. Dasar-dasar Elemen <audio>

Berikut contoh sederhana penggunaan elemen <audio>:

HTML 
<audio controls>
  <source src="audio/contoh.mp3" type="audio/mp3">
  Browser Anda tidak mendukung elemen audio.
</audio>

2. Atribut Penting <audio>

  • controls: Menampilkan kontrol pemutaran audio (play, pause, volume, dll)
  • autoplay: Audio akan otomatis diputar saat halaman dimuat
  • loop: Audio akan diulang secara otomatis setelah selesai
  • muted: Audio akan dimulai dalam keadaan mute
  • preload: Mengontrol pra-pemuatan audio (auto, metadata, none)

3. Beberapa Sumber Audio

Anda dapat menyediakan beberapa format audio untuk kompatibilitas browser yang lebih baik:

HTML 
<audio controls>
  <source src="audio/contoh.mp3" type="audio/mp3">
  <source src="audio/contoh.ogg" type="audio/ogg">
  <source src="audio/contoh.wav" type="audio/wav">
  Browser Anda tidak mendukung elemen audio HTML5.
  <a href="audio/contoh.mp3">Unduh audio</a> sebagai gantinya.
</audio>
Tip: Format MP3 didukung oleh hampir semua browser modern. Gunakan format ini sebagai format utama dan sertakan format lain sebagai cadangan.

4. Audio dengan Autoplay dan Loop

Gunakan atribut autoplay dan loop untuk pemutaran otomatis dan berulang:

HTML 
<audio autoplay loop>
  <source src="audio/lagu-latar.mp3" type="audio/mp3">
  Browser Anda tidak mendukung elemen audio.
</audio>
Peringatan: Penggunaan autoplay mungkin tidak berfungsi di beberapa browser karena kebijakan autoplay yang ketat. Biasanya, audio hanya akan otomatis diputar jika pengguna telah berinteraksi dengan halaman terlebih dahulu.

5. Mengontrol Audio dengan JavaScript

Anda dapat mengontrol pemutaran audio menggunakan JavaScript:

HTML 
<audio id="myAudio">
  <source src="audio/efek-suaraku.mp3" type="audio/mp3">
</audio>

<button onclick="document.getElementById('myAudio').play()">Putar</button>
<button onclick="document.getElementById('myAudio').pause()">Jeda</button>
<button onclick="document.getElementById('myAudio').volume += 0.1">Volume +</button>
<button onclick="document.getElementById('myAudio').volume -= 0.1">Volume -</button>

6. Menyembunyikan Kontrol Audio

Untuk audio latar belakang, Anda bisa menyembunyikan kontrolnya:

HTML 
<audio autoplay loop style="display: none;">
  <source src="audio/lagu-latar.mp3" type="audio/mp3">
  Browser Anda tidak mendukung elemen audio.
</audio>
Penting: Selalu pertimbangkan pengalaman pengguna saat menggunakan audio otomatis. Beri pengguna kendali untuk mematikan atau menyesuaikan volume audio.

7. Format Audio yang Direkomendasikan

Format MIME Type Keterangan
MP3 audio/mp3 Format paling populer, didukung oleh hampir semua browser
WAV audio/wav Kualitas tinggi, tetapi ukuran file besar
OGG audio/ogg Format terbuka dengan kompresi yang baik
Tip Lanjutan: Untuk pengalaman audio yang lebih kaya, pertimbangkan untuk menggunakan Web Audio API yang menyediakan kontrol yang lebih canggih atas pemrosesan dan sintesis audio.
