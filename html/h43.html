Menambahkan Gambar dengan <img>

Memahami Tag <img> dalam HTML

Tag <img> digunakan untuk menambahkan gambar ke dalam halaman web. Berbeda dengan elemen lain, tag ini tidak memiliki tag penutup dan termasuk dalam kategori void element.

1. Dasar Penggunaan <img>

Berikut adalah contoh dasar penggunaan tag <img>:

HTML 
<!-- Gambar dengan path relatif -->
<img src="gambar/logo.png" alt="Logo Perusahaan">

<!-- Gambar dengan URL absolut -->
<img src="https://contoh.com/gambar/logo.png" alt="Logo">

<!-- Gambar dengan lebar dan tinggi -->
<img src="foto.jpg" alt="Foto Profil" width="200" height="150">

2. Atribut Wajib

src (source)

Menentukan lokasi file gambar. Bisa berupa path relatif atau URL absolut.

HTML 
<!-- Path relatif -->
<img src="images/logo.png" alt="Logo">

<!-- URL absolut -->
<img src="https://contoh.com/gambar/header.jpg" alt="Header">
alt (alternative text)

Menyediakan teks alternatif yang akan ditampilkan jika gambar tidak dapat dimuat.

HTML 
<img src="kucing.jpg" alt="Kucing lucu sedang tidur">

3. Atribut Penting Lainnya

width dan height

Mengatur lebar dan tinggi gambar dalam piksel.

HTML 
<!-- Atur lebar dan tinggi -->
<img src="foto.jpg" alt="Foto" width="300" height="200">

<!-- Hanya atur lebar, tinggi menyesuaikan aspek rasio -->
<img src="foto.jpg" alt="Foto" width="300">
loading

Mengontrol kapan gambar akan dimuat.

HTML 
<!-- Gambar dimuat saat terlihat di viewport -->
<img src="gambar-besar.jpg" alt="Gambar Besar" loading="lazy">

4. Gambar Responsif

Menggunakan CSS
HTML 
<style>
  .gambar-responsif {
    max-width: 100%;
    height: auto;
  }
</style>

<img src="gambar.jpg" alt="Gambar Responsif" class="gambar-responsif">
Menggunakan srcset dan sizes
HTML 
<img 
  src="gambar-kecil.jpg" 
  srcset="gambar-kecil.jpg 300w,
          gambar-sedang.jpg 600w,
          gambar-besar.jpg 1200w"
  sizes="(max-width: 600px) 300px,
         (max-width: 1200px) 600px,
         1200px"
  alt="Gambar Responsif"
  loading="lazy"
>

5. Gambar dengan Teks Alternatif yang Baik

HTML 
<!-- Tidak disarankan -->
<img src="chart.png" alt="Grafik">

<!-- Direkomendasikan -->
<img 
  src="chart.png" 
  alt="Grafik pertumbuhan penjualan triwulan 1-4 tahun 2023"
  title="Klik untuk memperbesar"
>

6. Gambar dengan Link

HTML 
<!-- Gambar yang dapat diklik -->
<a href="halaman-besar.html">
  <img 
    src="thumbnail.jpg" 
    alt="Lihat gambar ukuran penuh"
    width="200"
    height="150"
  >
</a>

<!-- Gambar dengan tooltip -->
<img 
  src="info.png" 
  alt="Informasi"
  title="Klik untuk informasi lebih lanjut"
  style="cursor: help;"
>

7. Format Gambar yang Didukung

Format Ekstensi Kegunaan Dukungan
JPEG .jpg, .jpeg Foto ✓
PNG .png Grafik dengan transparansi ✓
GIF .gif Animasi sederhana ✓
SVG .svg Grafik vektor ✓
WebP .webp Format modern dengan kompresi lebih baik ✓ (kecuali IE)
AVIF .avif Format gambar modern ✓ (terbatas)

8. Praktik Terbaik

  • Selalu sertakan atribut alt yang deskriptif
  • Optimalkan ukuran file gambar untuk web
  • Gunakan format gambar yang tepat untuk konten yang sesuai
  • Pertimbangkan untuk menggunakan gambar responsif
  • Gunakan loading="lazy" untuk gambar di bawah fold
  • Pertimbangkan penggunaan CDN untuk gambar jika situs Anda besar
  • Gunakan teks alternatif yang bermakna untuk gambar konten
  • Hindari teks dalam gambar (gunakan HTML/CSS untuk teks)
