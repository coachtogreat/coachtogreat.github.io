Attribute Selector CSS: [type="text"] dan Lainnya

Attribute selector adalah selector CSS yang memungkinkan Anda memilih elemen berdasarkan atribut atau nilai atribut tertentu. Ini sangat berguna ketika Anda ingin menargetkan elemen dengan atribut spesifik tanpa perlu menambahkan class atau ID tambahan.

1. Dasar Attribute Selector

[attribute]

Memilih elemen yang memiliki atribut tertentu, terlepas dari nilainya.

CSS 
/* Memilih semua elemen yang memiliki atribut 'target' */
a[target] {
  color: #e91e63;
  text-decoration: none;
  font-weight: 500;
}

/* Memilih semua gambar yang memiliki atribut 'alt' */
img[alt] {
  border: 2px solid #4caf50;
  border-radius: 4px;
}

/* Memilih semua input yang memiliki atribut 'required' */
input[required] {
  border-left: 3px solid #ff5722;
  padding-left: 8px;
}
Contoh [attribute]:

Link dengan target _blank

Link biasa (tanpa target)

2. Attribute Selector dengan Nilai Spesifik

[attribute="value"]

Memilih elemen dengan atribut yang persis sama dengan nilai yang ditentukan.

CSS 
/* Memilih input dengan type="submit" */
input[type="submit"] {
  background-color: #4caf50;
  color: white;
  border: none;
  padding: 8px 16px;
  border-radius: 4px;
  cursor: pointer;
}

/* Memilih link dengan target="_blank" */
a[target="_blank"]::after {
  content: " â†—";
  font-size: 0.8em;
  opacity: 0.7;
}

/* Memilih elemen dengan data-status="completed" */
[data-status="completed"] {
  text-decoration: line-through;
  color: #9e9e9e;
}
Contoh [attribute="value"]:
  • Membeli bahan makanan
  • Menyelesaikan laporan
  • Menelepon klien

3. Attribute Selector dengan Pencocokan Parsial

[attribute^="value"] (Dimulai dengan)

Memilih elemen yang nilai atributnya dimulai dengan nilai tertentu.

[attribute$="value"] (Diakhiri dengan)

Memilih elemen yang nilai atributnya diakhiri dengan nilai tertentu.

[attribute*="value"] (Mengandung)

Memilih elemen yang nilai atributnya mengandung nilai tertentu di mana saja.

CSS 
/* Link yang dimulai dengan "https://" */
a[href^="https://"] {
  color: #4caf50;
  font-weight: 500;
}

/* Link yang diakhiri dengan ".pdf" */
a[href$=".pdf"]::after {
  content: " (PDF)";
  font-size: 0.8em;
  color: #f44336;
}

/* Gambar dengan "logo" di mana saja di src */
img[src*="logo"] {
  border: 2px solid #9c27b0;
  border-radius: 50%;
  padding: 5px;
}

/* Class yang diakhiri dengan "-primary" */
[class$="-primary"] {
  background-color: #2196f3;
  color: white;
}

/* Class yang mengandung "btn" */
[class*="btn"] {
  display: inline-block;
  padding: 8px 16px;
  margin: 4px;
  border-radius: 4px;
  text-decoration: none;
  cursor: pointer;
  transition: background-color 0.3s;
}
Contoh Pencocokan Parsial:
Kunjungi Website Unduh PDF

Gambar dengan "logo" di URL

4. Attribute Selector dengan Pencocokan Case-Insensitive

[attribute="value" i]

Memilih elemen dengan nilai atribut yang cocok, tidak memperhatikan huruf besar/kecil.

CSS 
/* Cocok dengan "success", "SUCCESS", "Success", dll */
[data-status="success" i] {
  background-color: #e8f5e9;
  color: #2e7d32;
  padding: 0.5rem;
  border-left: 3px solid #4caf50;
}

/* Cocok dengan "error", "ERROR", "Error", dll */
[data-status="error" i] {
  background-color: #ffebee;
  color: #c62828;
  padding: 0.5rem;
  border-left: 3px solid #f44336;
}
Contoh Case-Insensitive Matching:
Status: Sukses (case-sensitive)
Status: SUKSES (uppercase)
Status: Success (title case)
Status: Error (case-sensitive)
Status: ERROR (uppercase)

5. Attribute Selector dengan Pencocokan Kata Utuh

[attribute~="value"]

Memilih elemen dengan atribut yang berisi kata tertentu yang dipisahkan spasi.

[attribute|="value"]

Memilih elemen dengan atribut yang dimulai dengan kata yang diberikan, diikuti oleh tanda hubung (-).

CSS 
/* Memilih elemen dengan class yang berisi kata "warning" */
[class~="warning"] {
  background-color: #fff8e1;
  border-left: 4px solid #ffc107;
  padding: 1rem;
  margin: 0.5rem 0;
}

/* Memilih elemen dengan hreflang yang dimulai dengan "en" */
[hreflang|="en"] {
  border: 1px solid #2196f3;
  padding: 0.25rem 0.5rem;
  display: inline-block;
  margin: 0.25rem;
  border-radius: 3px;
  font-size: 0.8em;
}
Contoh Pencocokan Kata Utuh:
Ini adalah peringatan penting!
Ini pesan biasa

Kesimpulan

Attribute selector adalah fitur CSS yang sangat kuat yang memungkinkan Anda memilih elemen berdasarkan atribut atau nilai atributnya. Dengan menguasai berbagai jenis attribute selector, Anda dapat menulis CSS yang lebih efisien dan meminimalkan kebutuhan akan class tambahan.

Beberapa keuntungan menggunakan attribute selector:

  • Spesifisitas yang lebih rendah dibandingkan class dan ID
  • Kode yang lebih bersih dengan mengurangi kebutuhan class tambahan
  • Fleksibilitas dalam menargetkan elemen berdasarkan karakteristiknya
  • Pemeliharaan yang lebih mudah dengan pengelompokan gaya yang logis

Namun, perlu diingat bahwa beberapa attribute selector mungkin memiliki dukungan browser yang lebih terbatas dibandingkan selector dasar. Selalu uji kompatibilitas browser saat menggunakan selector yang lebih baru.

Dengan memahami dan menerapkan attribute selector dengan benar, Anda dapat menulis CSS yang lebih efisien dan mudah dikelola untuk proyek web Anda.
