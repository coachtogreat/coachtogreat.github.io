Memahami Grid Model

Panduan Lengkap tentang Grid Layout

Grid adalah sistem layout dua dimensi yang memungkinkan Anda membuat tata letak yang kompleks dengan mudah. Berbeda dengan Flexbox yang satu dimensi, Grid memungkinkan pengaturan kolom dan baris sekaligus, membuatnya sangat kuat untuk membuat desain web yang responsif.

1. Dasar-dasar Grid

Untuk memulai dengan Grid, Anda perlu mendefinisikan container sebagai grid:

CSS 
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr; /* 3 kolom dengan lebar yang sama */
  gap: 10px; /* Jarak antar item */
}

Keterangan:
  • display: grid - Mengubah elemen menjadi grid container
  • grid-template-columns - Mendefinisikan kolom grid
  • gap - Memberi jarak antar item grid
  • fr - Unit fraksi yang membagi ruang yang tersedia

2. Grid Template Areas

Anda dapat membuat layout yang lebih kompleks dengan menggunakan grid-template-areas:

CSS 
.container {
  display: grid;
  grid-template-areas:
    'header header header'
    'sidebar main main'
    'footer footer footer';
  gap: 10px;
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.footer { grid-area: footer; }

3. Responsive Grid

Buat grid yang responsif dengan mudah menggunakan fitur-fitur CSS Grid:

CSS 
.responsive-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  gap: 15px;
}

Penjelasan:
  • auto-fit - Secara otomatis menyesuaikan jumlah kolom
  • minmax(200px, 1fr) - Setiap kolom minimal 200px dan maksimal 1fr
  • Grid akan menyesuaikan jumlah kolom berdasarkan lebar layar

4. Grid vs Flexbox

Kapan sebaiknya menggunakan Grid dan kapan menggunakan Flexbox?

Gunakan CSS Grid ketika:
  • Membuat layout dua dimensi (baris dan kolom)
  • Membuat tata letak yang kompleks dengan area yang didefinisikan dengan jelas
  • Membutuhkan kontrol yang lebih ketat terhadap tata letak
Gunakan Flexbox ketika:
  • Mengatur item dalam satu dimensi (baris ATAU kolom)
  • Membutuhkan penjajaran konten yang lebih sederhana
  • Bekerja dengan komponen kecil dalam aplikasi

5. Contoh Praktis: Membuat Galeri Foto

Berikut contoh penerapan CSS Grid untuk membuat galeri foto yang responsif:

HTML 
<div class="gallery">
  <div class="gallery-item"><img src="foto1.jpg" alt="Foto 1"></div>
  <div class="gallery-item"><img src="foto2.jpg" alt="Foto 2"></div>
  <div class="gallery-item"><img src="foto3.jpg" alt="Foto 3"></div>
  <!-- tambahkan lebih banyak item -->
</div>
CSS 
.gallery {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
  gap: 15px;
  padding: 15px;
}

.gallery-item {
  overflow: hidden;
  border-radius: 8px;
  box-shadow: 0 2px 5px rgba(0,0,0,0.1);
}

.gallery-item img {
  width: 100%;
  height: 200px;
  object-fit: cover;
  transition: transform 0.3s ease;
}

.gallery-item:hover img {
  transform: scale(1.05);
}
Best Practices CSS Grid:
  • Gunakan grid-template-areas untuk layout yang kompleks
  • Manfaatkan minmax() untuk grid yang responsif
  • Gunakan gap untuk spasi yang konsisten
  • Kombinasikan dengan Flexbox untuk hasil terbaik
  • Selalu uji di berbagai ukuran layar
