Tag Heading HTML

Dalam HTML, tag heading (<h1> sampai <h6>) digunakan untuk mendefinisikan judul dan subjudul pada halaman web. Tag ini menciptakan struktur hierarkis, dengan <h1> sebagai yang paling penting (judul utama) dan <h6> sebagai yang paling tidak penting.

Tag Heading dalam Dokumen Ini:

  • <h2 class="intro-title"> - Ini adalah heading sekunder (level 2), digunakan untuk judul utama
  • <h3 class="section-title"> - Ini adalah heading tersier (level 3), digunakan untuk judul bagian
  • <h4> - Ini adalah heading kuaterner (level 4), digunakan untuk subbagian

Poin Penting Tentang Heading HTML:

  • Mesin pencari menggunakan heading untuk mengindeks struktur dan konten halaman web
  • Pengguna sering kali melihat sekilas halaman berdasarkan heading-nya

  • Jangan menggunakan heading hanya untuk membuat teks lebih besar atau tebal - gunakan CSS untuk keperluan penataan gaya

Dalam dokumen ini, Anda menggunakan tag heading dengan tepat untuk menciptakan hierarki konten yang jelas, yang sangat baik untuk SEO dan aksesibilitas. Kelas-kelas (intro-title, section-title) juga memungkinkan penataan gaya yang konsisten melalui CSS.
