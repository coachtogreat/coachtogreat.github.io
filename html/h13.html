Pengenalan HTML

Contoh Kode HTML 5

HTML 
<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Profil Saya</title>
</head>
<body>

  <header>
    <h1>Selamat Datang di Profil Saya</h1>
    <nav>
      <ul>
        <li><a href="#tentang">Tentang</a></li>
        <li><a href="#hobi">Hobi</a></li>
        <li><a href="#kontak">Kontak</a></li>
      </ul>
    </nav>
  </header>

  <main>
    <section id="tentang">
      <h2>Tentang Saya</h2>
      <p>Nama saya Rachman. Saya seorang pengembang web yang suka membuat aplikasi menggunakan HTML, CSS, dan JavaScript.</p>
    </section>

    <section id="hobi">
      <h2>Hobi</h2>
      <ul>
        <li>Ngoding</li>
        <li>Membaca</li>
        <li>Jalan-jalan</li>
      </ul>
    </section>
  </main>

  <footer id="kontak">
    <h2>Kontak</h2>
    <p>Email: <a href="mailto:andi@example.com">andi@example.com</a></p>
  </footer>

</body>
</html>

Penjelasan per Detail Per Elemen

1. <!DOCTYPE html>

Menandakan bahwa ini adalah dokumen HTML5. Harus selalu berada di bagian paling atas file HTML.

2. <html lang="id">

Tag pembuka untuk seluruh dokumen HTML.
Atribut lang="id" memberitahu browser dan mesin pencari bahwa halaman ini berbahasa Indonesia.

3. <head>...</head>

Bagian ini tidak terlihat langsung oleh pengguna, tapi penting untuk pengaturan halaman.

  • <meta charset="UTF-8">: Memastikan halaman bisa menampilkan karakter dari berbagai bahasa dengan benar.
  • <meta name="viewport"...>: Membuat tampilan halaman dapat menyesuaikan ukuran layar perangkat (responsif).
  • <title> Profil Saya</title>: Menentukan teks yang muncul di tab browser.

4. <body>...</body>

Ini adalah bagian utama halaman web. Semua konten yang terlihat oleh pengguna berada di sini.

<header>...</header>

Bagian atas halaman, biasanya berisi judul utama dan navigasi.
Isi:

  • <h1>: Judul besar halaman (penting untuk SEO).
  • <nav>: Navigasi atau menu ke bagian lain.
  • <ul> unordered list): Daftar tanpa urutan.
  • <li>: Item di dalam daftar.
  • <a href="#id">: Tautan ke bagian dalam halaman yang memiliki ID tertentu.

<main>...</main>

Bagian utama konten. Biasanya terdiri dari beberapa section.

<section id="tentang">

  • <h2>: Subjudul bagian
  • <p>: Paragraph teks

<section id="hobi">

  • <ul>: Daftar hobi dalam format tidak berurutan.

<section id="footer">

Bagian paling bawah halaman, biasanya berisi:

  • Informasi kontak
  • Hak cipta (copyright)
  • Tautan social media
  • Tautan mailto: akan membuka aplikasi email saat diklik

Tips Tambahan

Berikut adalah beberapa elemen HTML dasar dan fungsinya:

Elemen Fungsi Keterangan
<h1> - <h6> Heading Gunakan satu <h1> per halaman, lalu <h2>, <h3>, dst untuk sub-judul.
<p> Paragraf Untuk teks biasa.
<ul>, <ol>, <li> Daftar Unordered (ul) atau ordered (ol).
<a href=""> Tautan Untuk navigasi ke halaman lain atau bagian lain dalam halaman.
<img src="..." alt="..."> Gambar Gunakan atribut alt untuk deskripsi alternatif.
<div>, <span> Elemen kontainer Digunakan untuk pengelompokan dan styling (CSS).
