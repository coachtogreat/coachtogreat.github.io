Textarea & Select Dropdown

Mengelola Input Teks Panjang dan Pilihan dalam Formulir

Halaman ini akan membahas secara mendalam tentang elemen <textarea> dan <select> yang sangat berguna untuk mengumpulkan input teks panjang dan pilihan dari pengguna.

1. Textarea (<textarea>)

Elemen untuk input teks multi-baris seperti komentar, deskripsi, atau pesan panjang.

Contoh Penggunaan Dasar:
HTML 
<label for="komentar">Komentar Anda:</label>
<textarea id="komentar" name="komentar" rows="4" 
          placeholder="Tulis komentar Anda di sini..."></textarea>
Atribut Penting Textarea:
Atribut Deskripsi Contoh
rows Jumlah baris yang terlihat rows="4"
cols Lebar dalam karakter (sebaiknya gunakan CSS) cols="50"
maxlength Jumlah karakter maksimum maxlength="500"
minlength Jumlah karakter minimum minlength="10"
placeholder Teks petunjuk dalam textarea placeholder="Tulis pesan..."
readonly Hanya bisa dibaca readonly
disabled Menonaktifkan textarea disabled
wrap Penggulungan teks (soft/hard) wrap="hard"
Contoh Textarea Lanjutan:
Minimal 10 karakter, maksimal 200 karakter
HTML 
<label for="alamat">Alamat Lengkap:</label>
<textarea id="alamat" name="alamat" rows="3" 
          placeholder="Jl. Contoh No. 123, Kota" 
          required minlength="10" maxlength="200"
          style="width: 100%;"></textarea>
<small>Minimal 10 karakter, maksimal 200 karakter</small>
Tip: Gunakan CSS resize: vertical atau resize: none untuk mengontrol apakah pengguna dapat mengubah ukuran textarea.

2. Select Dropdown (<select>)

Elemen untuk memilih satu atau beberapa opsi dari daftar dropdown.

Select Dasar (Pilihan Tunggal):
HTML 
<label for="provinsi">Pilih Provinsi:</label>
<select id="provinsi" name="provinsi" required>
  <option value="" disabled selected>-- Pilih Provinsi --</option>
  <option value="jabar">Jawa Barat</option>
  <option value="jatim">Jawa Timur</option>
  <option value="jateng">Jawa Tengah</option>
  <option value="banten">Banten</option>
  <option value="dki">DKI Jakarta</option>
</select>
Atribut Penting Select:
Atribut Deskripsi Contoh
multiple Memungkinkan banyak pilihan multiple
size Jumlah opsi yang terlihat size="5"
required Harus dipilih required
disabled Menonaktifkan select disabled
Atribut Penting Option:
Atribut Deskripsi Contoh
value Nilai yang dikirim ke server value="jabar"
selected Opsi terpilih secara default selected
disabled Menonaktifkan opsi disabled
Select dengan Grup Opsi:
HTML 
<label for="makanan">Makanan Favorit:</label>
<select id="makanan" name="makanan">
  <optgroup label="Makanan Indonesia">
    <option value="nasi-goreng">Nasi Goreng</option>
    <option value="sate">Sate</option>
    <option value="rendang">Rendang</option>
  </optgroup>
  <optgroup label="Makanan Barat">
    <option value="pizza">Pizza</option>
    <option value="burger">Burger</option>
    <option value="pasta">Pasta</option>
  </optgroup>
</select>
Select dengan Multiple Pilihan:
Tahan tombol Ctrl (Windows/Linux) atau Cmd (Mac) untuk memilih beberapa opsi
HTML 
<label for="hobi">Pilih Hobi (Tekan Ctrl/Cmd untuk memilih banyak):</label>
<select id="hobi" name="hobi" multiple size="4">
  <option value="membaca">Membaca</option>
  <option value="menulis">Menulis</option>
  <option value="berenang">Berenang</option>
  <option value="bersepeda">Bersepeda</option>
  <option value="musik">Bermusik</option>
  <option value="traveling">Traveling</option>
</select>
<small>Tahan tombol Ctrl (Windows/Linux) atau Cmd (Mac) untuk memilih beberapa opsi</small>
DataList untuk Autocomplete

Anda juga dapat menggunakan <datalist> untuk membuat input dengan saran otomatis:

HTML 
<label for="browser">Pilih browser favorit Anda:</label>
<input list="browsers" id="browser" name="browser">
<datalist id="browsers">
  <option value="Google Chrome">
  <option value="Mozilla Firefox">
  <option value="Microsoft Edge">
  <option value="Safari">
  <option value="Opera">
</datalist>

Praktik Terbaik Penggunaan Textarea dan Select

  • Gunakan <textarea> untuk input teks panjang (lebih dari satu baris)
  • Gunakan <select> saat ada lebih dari 5 opsi untuk menghemat ruang
  • Urutkan opsi dalam <select> secara logis (abjad, numerik, atau berdasarkan prioritas)
  • Gunakan <optgroup> untuk mengelompokkan opsi yang terkait
  • Hindari menggunakan multiple dengan lebih dari 10 opsi yang dapat dipilih
  • Gunakan atribut required untuk bidang yang wajib diisi
  • Berikan teks bantuan atau contoh format yang diharapkan
  • Uji formulir pada berbagai perangkat dan browser
