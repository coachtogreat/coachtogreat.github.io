Atribut Tabel HTML

Memahami Atribut-atribut Tabel dalam HTML

Tabel HTML memiliki berbagai atribut yang memungkinkan Anda mengontrol tampilan dan perilakunya. Beberapa atribut sudah tidak digunakan lagi (deprecated) dan digantikan dengan CSS, sementara yang lain masih relevan digunakan.

Catatan: Atribut yang ditandai dengan coretan sudah tidak direkomendasikan penggunaannya dan sebaiknya diganti dengan CSS.

1. Atribut untuk Tag <table>

Atribut Deskripsi Nilai yang Mungkin Status
border Menentukan ketebalan border tabel angka (contoh: 1, 2) Deprecated (gunakan CSS border)
cellpadding Jarak antara isi sel dengan border-nya angka (dalam pixel) Deprecated (gunakan CSS padding)
cellspacing Jarak antar sel dalam tabel angka (dalam pixel) Deprecated (gunakan CSS border-spacing)
width Lebar tabel pixel atau persentase Deprecated (gunakan CSS width)
height Tinggi tabel pixel atau persentase Deprecated (gunakan CSS height)
align Posisi horizontal tabel left, center, right Deprecated (gunakan CSS margin: 0 auto)
bgcolor Warna latar belakang tabel nama warna atau kode hex Deprecated (gunakan CSS background-color)
rules Menampilkan garis pemisah sel none, groups, rows, cols, all Masih didukung
summary Ringkasan isi tabel (untuk aksesibilitas) teks Deprecated (gunakan <caption> atau aria-describedby)
Contoh Penggunaan:
HTML 
<!-- Cara lama (tidak direkomendasikan) -->
<table border="1" cellpadding="5" cellspacing="0" width="100%" bgcolor="#f0f0f0">
  <!-- isi tabel -->
</table>

<!-- Cara modern dengan CSS -->
<table style="border: 1px solid #000; border-collapse: collapse; width: 100%; background-color: #f0f0f0;">
  <caption>Judul Tabel</caption>
  <!-- isi tabel -->
</table>

2. Atribut untuk Tag <tr>, <th>, dan <td>

Atribut Deskripsi Nilai yang Mungkin Status
rowspan Menggabungkan beberapa baris angka (jumlah baris yang digabung) Masih didukung
colspan Menggabungkan beberapa kolom angka (jumlah kolom yang digabung) Masih didukung
headers Menghubungkan sel data dengan header terkait ID dari elemen header Masih didukung (aksesibilitas)
scope Menentukan cakupan header row, col, rowgroup, colgroup Masih didukung (aksesibilitas)
abbr Versi singkat dari konten sel teks Masih didukung (aksesibilitas)
width Lebar sel pixel atau persentase Deprecated (gunakan CSS)
height Tinggi sel pixel atau persentase Deprecated (gunakan CSS)
bgcolor Warna latar belakang sel nama warna atau kode hex Deprecated (gunakan CSS)
align Posisi horizontal konten left, center, right, justify Deprecated (gunakan CSS)
valign Posisi vertikal konten top, middle, bottom, baseline Deprecated (gunakan CSS)
Contoh Penggunaan:
HTML 
<table border="1">
  <tr>
    <th scope="col">Nama</th>
    <th scope="col">Nilai</th>
  </tr>
  <tr>
    <td>Matematika</td>
    <td align="center">90</td>
  </tr>
  <tr>
    <td>Bahasa Indonesia</td>
    <td align="center">85</td>
  </tr>
  <tr>
    <td colspan="2" bgcolor="#f0f0f0">Rata-rata: 87.5</td>
  </tr>
</table>

3. Atribut untuk <colgroup> dan <col>

Atribut Deskripsi Nilai yang Mungkin Status
span Jumlah kolom yang akan diformat angka Masih didukung
width Lebar kolom pixel, persentase, atau relatif (*) Deprecated (gunakan CSS)
align Posisi horizontal konten left, center, right, justify Deprecated (gunakan CSS)
valign Posisi vertikal konten top, middle, bottom, baseline Deprecated (gunakan CSS)
Contoh Penggunaan:
HTML 
<table border="1">
  <colgroup>
    <col style="background-color:#f0f0f0">
    <col span="2" style="background-color:#e0e0e0">
  </colgroup>
  <tr>
    <th>No.</th>
    <th>Nama Barang</th>
    <th>Harga</th>
  </tr>
  <tr>
    <td>1</td>
    <td>Buku Tulis</td>
    <td>Rp 5.000</td>
  </tr>
</table>

4. Atribut Global yang Dapat Digunakan

Selain atribut khusus tabel, elemen tabel juga mendukung atribut global HTML5:

Atribut Deskripsi Contoh
id Pengidentifikasi unik untuk elemen id="tabel-data"
class Nama kelas untuk styling CSS class="tabel-striped"
style CSS inline style="width: 100%;"
title Informasi tambahan (tooltip) title="Data per 2023"
data-* Atribut data kustom data-sortable="true"
hidden Menyembunyikan elemen hidden

5. Atribut Aksesibilitas

Berikut beberapa atribut yang meningkatkan aksesibilitas tabel:

Atribut Deskripsi Contoh
scope Menentukan cakupan header (baris/kolom) <th scope="col">Nama</th>
headers Menghubungkan sel data dengan header headers="nama-header"
aria-label Label untuk pembaca layar aria-label="Tabel Data Mahasiswa"
aria-describedby Menghubungkan dengan deskripsi tabel aria-describedby="tabel-desc"
role Peran ARIA untuk aksesibilitas role="table", role="row", dll.
Contoh Tabel dengan Aksesibilitas:
HTML 
<table role="table" aria-label="Daftar Harga Produk">
  <caption>Daftar Harga Produk Terbaru</caption>
  <thead>
    <tr role="row">
      <th scope="col" role="columnheader">No.</th>
      <th scope="col" role="columnheader">Nama Produk</th>
      <th scope="col" role="columnheader">Harga (Rp)</th>
    </tr>
  </thead>
  <tbody>
    <tr role="row">
      <td role="cell">1</td>
      <td role="cell">Laptop ASUS X441NA</td>
      <td role="cell">4.499.000</td>
    </tr>
  </tbody>
</table>

6. Praktik Terbaik

  • Gunakan CSS untuk styling, bukan atribut HTML yang sudah tidak didukung
  • Selalu sertakan <th> untuk header tabel
  • Gunakan scope pada header untuk aksesibilitas
  • Pertimbangkan penggunaan <caption> untuk judul tabel
  • Gunakan rowspan dan colspan dengan bijak
  • Pastikan tabel responsif di perangkat mobile
  • Uji aksesibilitas tabel dengan pembaca layar
  • Hindari tabel bersarang yang terlalu kompleks
