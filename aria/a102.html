Daftar Istilah WAI-ARIA

A

Istilah Awal A 
// ARIA (Accessible Rich Internet Applications)
- Framework untuk meningkatkan aksesibilitas web

// Accessibility
- Kemudahan akses pengguna dengan keterbatasan

// Assistive Technology
- Teknologi yang membantu pengguna dengan keterbatasan

C

  • ChromeVox: Screen reader untuk Chrome
  • Chrome DevTools: Alat pengembangan Chrome
  • Code: Kode program atau perangkat lunak

D

Istilah Awal D 
// Developer Tools
- Alat untuk mengembangkan dan debugging web

// Documentation
- Dokumentasi resmi dan panduan penggunaan

// DOM (Document Object Model)
- Struktur dokumen web

E

  • Error: Pesan kesalahan atau masalah
  • Event: Peristiwa yang terjadi di web
  • Element: Komponen atau bagian dari halaman web

F

Istilah Awal F 
// Focus: Elemen yang aktif atau dipilih

// Form: Komponen untuk mengumpulkan data

// Function: Blok kode yang melakukan tugas tertentu

G

  • Guideline: Panduan atau standar
  • Grid: Struktur tabel atau grid
  • Group: Kumpulan elemen yang terkait

H

Istilah Awal H 
// HTML: Hypertext Markup Language
- Bahasa markup untuk web

// Header: Bagian atas halaman

// Help: Bantuan atau dokumentasi

I

  • Implementation: Implementasi atau penerapan
  • Integration: Integrasi atau penggabungan
  • Interface: Antarmuka pengguna

J

Istilah Awal J 
// JavaScript: Bahasa pemrograman web

// JAWS: Screen reader untuk Windows

// JSON: Format data JavaScript Object Notation

K

  • Keyboard: Papan ketik
  • Key: Kunci atau akses
  • Keyboard Navigation: Navigasi menggunakan papan ketik

L

Istilah Awal L 
// Label: Label atau keterangan

// Landmark: Area penting dalam halaman

// Library: Kumpulan fungsi atau komponen

M

  • Menu: Daftar pilihan atau opsi
  • Modal: Dialog atau jendela modal
  • Mobile: Perangkat mobile atau seluler

N

Istilah Awal N 
// Navigation: Navigasi atau perpindahan

// NVDA: Screen reader gratis untuk Windows

// Node: Elemen dalam struktur DOM

O

  • Optimization: Optimasi atau peningkatan
  • Output: Hasil atau keluaran
  • Operator: Operator atau simbol

P

Istilah Awal P 
// Property: Properti atau atribut

// Protocol: Protokol atau standar

// Pattern: Pola atau template

Q

  • Query: Pertanyaan atau pencarian
  • Quality: Kualitas atau mutu
  • Quick: Cepat atau singkat

R

Istilah Awal R 
// Role: Peran atau fungsi

// Repository: Repositori atau gudang

// Response: Respons atau jawaban

S

  • Screen Reader: Pembaca layar
  • Script: Skrip atau kode program
  • Selector: Selektor atau pemilih

T

Istilah Awal T 
// Testing: Pengujian atau tes

// Tool: Alat atau utilitas

// Template: Template atau pola

U

  • UI (User Interface): Antarmuka pengguna
  • UX (User Experience): Pengalaman pengguna
  • Update: Pembaruan atau upgrade

V

Istilah Awal V 
// Validation: Validasi atau pengecekan

// Value: Nilai atau harga

// Version: Versi atau edisi

W

  • WCAG: Web Content Accessibility Guidelines
  • WebAIM: Web Accessibility in Mind
  • WAI: Web Accessibility Initiative

X

Istilah Awal X 
// XML: Extensible Markup Language

// XPath: Path untuk mencari elemen

// XSLT: XML Stylesheet Language Transformations

Y

  • Year: Tahun atau periode
  • Yes: Ya atau setuju
  • Yield: Hasil atau output

Z

Istilah Awal Z 
// Zone: Zona atau area

// Zoom: Perbesar atau perkecil

// Zero: Nol atau kosong

Kesimpulan

Daftar istilah ini membantu memahami terminologi yang digunakan dalam WAI-ARIA dan pengembangan web aksesibel. Dengan memahami istilah-istilah ini, pengembang dapat lebih mudah mengikuti dokumentasi dan best practices dalam implementasi WAI-ARIA.
