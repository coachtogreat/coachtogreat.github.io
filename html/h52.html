Daftar Tidak Berurutan (Unordered List) HTML

Memahami Tag <ul> dan <li> dalam HTML

Daftar tidak berurutan (unordered list) digunakan untuk menampilkan daftar item yang tidak memiliki urutan atau prioritas tertentu. Setiap item dalam daftar biasanya ditandai dengan bullet point secara default.

1. Dasar Penggunaan <ul>

Berikut adalah contoh dasar penggunaan tag <ul>:

HTML 
<ul>
  <li>Apel</li>
  <li>Jeruk</li>
  <li>Mangga</li>
</ul>
Hasil:
  • Apel
  • Jeruk
  • Mangga

2. Jenis Bullet Point

Anda dapat mengubah gaya bullet point menggunakan atribut type (meskipun penggunaan CSS lebih direkomendasikan):

HTML 
<ul type="disc">...</ul> <!-- Bulat penuh (default) -->
<ul type="circle">...</ul> <!-- Lingkaran kosong -->
<ul type="square">...</ul> <!-- Kotak -->
<ul style="list-style-type: none;">...</ul> <!-- Tanpa bullet -->
Contoh:
Bulat Penuh (default):
  • Item 1
  • Item 2
Lingkaran Kosong:
  • Item 1
  • Item 2
Kotak:
  • Item 1
  • Item 2
Tanpa Bullet:
  • Item 1
  • Item 2

3. Daftar Bersarang (Nested Lists)

Anda dapat membuat daftar di dalam daftar untuk membuat hierarki yang lebih kompleks:

HTML 
<ul>
  <li>Buah-buahan
    <ul>
      <li>Apel</li>
      <li>Jeruk</li>
    </ul>
  </li>
  <li>Sayuran
    <ul>
      <li>Bayam</li>
      <li>Wortel</li>
    </ul>
  </li>
</ul>
Hasil:
  • Buah-buahan
    • Apel
    • Jeruk
  • Sayuran
    • Bayam
    • Wortel

4. Menggabungkan dengan CSS

Anda dapat menyesuaikan tampilan daftar dengan CSS:

HTML 
<style>
  .daftar-kustom {
    list-style-type: none;
    padding-left: 0;
  }
  .daftar-kustom li {
    padding: 8px 0 8px 30px;
    position: relative;
    border-bottom: 1px solid #eee;
  }
  .daftar-kustom li::before {
    content: "→";
    color: #28a745;
    font-weight: bold;
    position: absolute;
    left: 0;
  }
  .daftar-kustom li:hover {
    background-color: #f8f9fa;
  }
</style>

<ul class="daftar-kustom">
  <li>Item pertama</li>
  <li>Item kedua</li>
  <li>Item ketiga</li>
</ul>
Hasil:
  • Item pertama
  • Item kedua
  • Item ketiga

5. Contoh Praktis

Berikut contoh penggunaan daftar tidak berurutan dalam menu navigasi:

HTML 
<style>
  .menu-navigasi {
    list-style-type: none;
    padding: 0;
    margin: 0;
    background-color: #343a40;
    overflow: hidden;
    border-radius: 5px;
  }
  .menu-navigasi li {
    float: left;
  }
  .menu-navigasi li a {
    display: block;
    color: white;
    text-align: center;
    padding: 14px 16px;
    text-decoration: none;
    transition: background-color 0.3s;
  }
  .menu-navigasi li a:hover {
    background-color: #495057;
  }
  .menu-navigasi li a.aktif {
    background-color: #28a745;
  }
</style>

<ul class="menu-navigasi">
  <li><a href="#beranda" class="aktif">Beranda</a></li>
  <li><a href="#tentang">Tentang</a></li>
  <li><a href="#layanan">Layanan</a></li>
  <li><a href="#kontak">Kontak</a></li>
</ul>
Hasil:
  • Beranda
  • Tentang
  • Layanan
  • Kontak

6. Praktik Terbaik

  • Gunakan daftar tidak berurutan untuk item yang tidak memiliki urutan atau prioritas tertentu
  • Hindari menggunakan daftar hanya untuk indentasi atau tata letak halaman
  • Gunakan CSS untuk mengontrol tampilan bullet point
  • Pastikan daftar tetap dapat dipahami tanpa CSS (aksesibilitas)
  • Gunakan daftar bersarang dengan bijak untuk menghindari hierarki yang terlalu dalam
  • Pertimbangkan penggunaan ikon sebagai pengganti bullet point untuk tampilan yang lebih menarik
