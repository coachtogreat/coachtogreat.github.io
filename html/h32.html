Teks dan Paragraf

Tag Paragraf HTML

Dalam HTML, tag <p> digunakan untuk membuat paragraf teks. Setiap paragraf secara otomatis memiliki jarak (margin) di atas dan di bawahnya, yang memisahkannya dari elemen-elemen lain di halaman web.

Contoh Penggunaan Tag Paragraf:

HTML 
<p>Ini adalah contoh paragraf pertama.</p>
<p>Ini adalah paragraf kedua. Perhatikan jarak otomatis antara kedua paragraf ini.</p>

Atribut Penting Tag Paragraf:

  • class - Menambahkan nama class untuk penataan gaya dengan CSS
  • id - Memberikan identifikasi unik untuk paragraf
  • style - Menambahkan gaya inline ke paragraf

Pemformatan Teks dalam Paragraf:

Anda bisa memformat teks dalam paragraf menggunakan berbagai tag seperti: <strong> untuk teks tebal, <em> untuk teks miring, dan <u> untuk teks bergaris bawah.

HTML 
<p>
  Ini adalah contoh teks dengan <strong>teks tebal</strong>, 
  <em>teks miring</em>, dan <u>teks bergaris bawah</u>.
</p>

Poin Penting Tentang Paragraf HTML:

  • Gunakan tag <br> untuk membuat baris baru dalam paragraf yang sama
  • Gunakan &nbsp; untuk menambahkan spasi yang tidak terputus
  • Hindari menggunakan tag <br> untuk membuat jarak antar paragraf
  • Gunakan CSS untuk mengatur jarak dan indentasi paragraf
