Pseudo-element CSS: ::before, ::after, dan Lainnya

Pseudo-element adalah kata kunci yang ditambahkan ke selector yang memungkinkan Anda menata bagian tertentu dari elemen yang dipilih. Berbeda dengan pseudo-class yang menargetkan state tertentu, pseudo-element memungkinkan Anda menambahkan elemen semu ke halaman tanpa perlu menambahkan elemen HTML tambahan.

1. ::before dan ::after

Pseudo-element ::before dan ::after memungkinkan Anda menyisipkan konten sebelum atau sesudah konten elemen.

CSS 
/* Menambahkan ikon sebelum elemen */
.before-demo::before {
  content: "→ ";
  color: #e91e63;
  font-weight: bold;
}

/* Menambahkan teks setelah elemen */
.after-demo::after {
  content: " ←";
  color: #2196f3;
  font-style: italic;
}

/* Menggunakan atribut data-* untuk konten dinamis */
[data-tooltip]::after {
  content: attr(data-tooltip);
  display: none;
  /* Tambahan styling untuk tooltip */
}
[data-tooltip]:hover::after {
  display: block;
  /* Tambahan styling untuk tooltip saat dihover */
}

2. ::first-line dan ::first-letter

Pseudo-element ini memungkinkan Anda menata baris pertama atau huruf pertama dari teks.

CSS 
/* Menata baris pertama teks */
.first-line-demo::first-line {
  font-weight: bold;
  color: #4caf50;
  text-transform: uppercase;
}

/* Menata huruf pertama teks (drop cap) */
.first-letter-demo::first-letter {
  font-size: 2.5em;
  float: left;
  line-height: 1;
  margin-right: 0.1em;
  color: #9c27b0;
  font-weight: bold;
}

3. ::selection

Memungkinkan Anda menata teks yang dipilih oleh pengguna.

CSS 
/* Mengubah warna teks yang dipilih */
.selection-demo::selection {
  background-color: #ffeb3b;
  color: #000;
  text-shadow: none;
}

/* Untuk seluruh dokumen */
::selection {
  background: #ff4081;
  color: white;
}
Contoh ::selection:

Coba pilih teks ini untuk melihat efek ::selection. Teks yang Anda pilih akan memiliki latar belakang kuning dengan teks hitam.

4. Pseudo-element Lainnya

::marker

Memungkinkan Anda menata bullet atau nomor pada daftar.

CSS 
/* Menata bullet/nomor pada daftar */
.marker-demo li::marker {
  color: #ff5722;
  font-weight: bold;
  content: "→ "; /* Beberapa browser mendukung penggantian konten */
}

::placeholder

Memungkinkan Anda menata placeholder pada input form.

CSS 
/* Menata teks placeholder */
.placeholder-demo::placeholder {
  color: #9e9e9e;
  font-style: italic;
  opacity: 1; /* Beberapa browser memerlukan ini */
}

/* Vendor prefix untuk kompatibilitas */
.placeholder-demo::-webkit-input-placeholder { /* Chrome/Opera/Safari */
  color: #9e9e9e;
  font-style: italic;
}
.placeholder-demo::-moz-placeholder { /* Firefox 19+ */
  color: #9e9e9e;
  font-style: italic;
  opacity: 1;
}
.placeholder-demo:-ms-input-placeholder { /* IE 10+ */
  color: #9e9e9e;
  font-style: italic;
}
.placeholder-demo:-moz-placeholder { /* Firefox 18- */
  color: #9e9e9e;
  font-style: italic;
}

Kesimpulan

Pseudo-element CSS memberikan fleksibilitas tambahan dalam menata elemen-elemen HTML tanpa perlu menambahkan markup tambahan. Dengan menggunakan pseudo-element seperti ::before, ::after, ::first-line, dan lainnya, Anda dapat menciptakan efek visual yang menarik dan meningkatkan pengalaman pengguna.

Beberapa hal penting yang perlu diingat:

  • Pseudo-element memerlukan properti content untuk ditampilkan (kecuali ::first-line dan ::first-letter)
  • Pseudo-element bersifat inline secara default
  • Beberapa pseudo-element seperti ::placeholder memerlukan vendor prefix untuk kompatibilitas browser yang lebih luas
  • Gunakan pseudo-element dengan bijak untuk meningkatkan aksesibilitas, bukan menguranginya

Dengan memahami dan memanfaatkan pseudo-element dengan baik, Anda dapat menciptakan desain yang lebih bersih, efisien, dan menarik.
