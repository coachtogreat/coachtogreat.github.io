Cara Menghubungkan CSS ke HTML

Ada tiga cara utama untuk menghubungkan CSS ke HTML: inline, internal (embedded), dan eksternal. Setiap metode memiliki kegunaan dan kasus penggunaannya masing-masing. Mari kita bahas masing-masing metode tersebut beserta kelebihan dan kekurangannya.

1. Inline CSS

Inline CSS adalah metode penulisan CSS langsung di dalam tag HTML menggunakan atribut style. Ini adalah cara yang paling spesifik dan memiliki prioritas tertinggi dalam CSS.

HTML 
<!DOCTYPE html>
<html>
  <head>
    <title>Contoh Inline CSS</title>
  </head>
  <body>
    <h1 style="color: blue; font-size: 24px;">Judul dengan Inline CSS</h1>
    <p style="color: #333; line-height: 1.6;">
      Ini adalah contoh paragraf dengan gaya inline CSS.
    </p>
    <div style="background-color: #f0f0f0; padding: 15px; border-radius: 5px;">
      <button style="background-color: #4CAF50; color: white; padding: 10px 15px; border: none; border-radius: 4px; cursor: pointer;">
        Tombol
      </button>
    </div>
  </body>
</html>

Kelebihan Inline CSS:

  • Mudah dan cepat untuk perubahan kecil
  • Memiliki spesifisitas tertinggi (akan mengalahkan CSS internal dan eksternal)
  • Berguna untuk pengujian cepat

Kekurangan Inline CSS:

  • Tidak efisien untuk styling banyak elemen
  • Membuat kode HTML berantakan
  • Sulit untuk pemeliharaan jangka panjang
  • Tidak bisa menggunakan fitur CSS yang lebih canggih seperti media queries

2. Internal CSS (Embedded CSS)

Internal CSS ditulis di dalam tag <style> yang biasanya ditempatkan di dalam bagian <head> dokumen HTML. Gaya ini hanya berlaku untuk satu halaman web.

HTML 
<!DOCTYPE html>
<html>
  <head>
    <title>Contoh Internal CSS</title>
    <style>
      /* Reset default margin dan padding */
      * {
        margin: 0;
        padding: 0;
        box-sizing: border-box;
      }

      /* Styling untuk body */
      body {
        font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
        line-height: 1.6;
        color: #333;
        max-width: 1200px;
        margin: 0 auto;
        padding: 20px;
      }

      /* Styling untuk header */
      header {
        background-color: #2c3e50;
        color: white;
        padding: 1rem 0;
        text-align: center;
        margin-bottom: 2rem;
        border-radius: 5px;
      }

      /* Styling untuk navigation */
      nav ul {
        list-style: none;
        display: flex;
        justify-content: center;
        gap: 2rem;
        padding: 1rem 0;
      }

      nav a {
        color: white;
        text-decoration: none;
        font-weight: 500;
        transition: color 0.3s;
      }

      nav a:hover {
        color: #3498db;
      }

      /* Styling untuk tombol */
      .btn {
        display: inline-block;
        background-color: #3498db;
        color: white;
        padding: 0.5rem 1rem;
        border: none;
        border-radius: 4px;
        cursor: pointer;
        text-decoration: none;
        transition: background-color 0.3s;
      }

      .btn:hover {
        background-color: #2980b9;
      }
    </style>
  </head>
  <body>
    <header>
      <h1>Website Saya</h1>
      <nav>
        <ul>
          <li><a href="#">Beranda</a></li>
          <li><a href="#">Tentang</a></li>
          <li><a href="#">Layanan</a></li>
          <li><a href="#">Kontak</a></li>
        </ul>
      </nav>
    </header>

    <main>
      <h2>Selamat Datang di Website Kami</h2>
      <p>Ini adalah contoh penggunaan internal CSS pada halaman web.</p>
      <a href="#" class="btn">Pelajari Lebih Lanjut</a>
    </main>
  </body>
</html>

Kelebihan Internal CSS:

  • Lebih terorganisir dibanding inline CSS
  • Dapat menggunakan semua fitur CSS termasuk media queries
  • Cocok untuk halaman tunggal yang membutuhkan gaya khusus
  • Mengurangi permintaan HTTP (tidak perlu file terpisah)

Kekurangan Internal CSS:

  • Hanya berlaku untuk satu halaman
  • Dapat membuat file HTML menjadi besar
  • Kurang efisien untuk situs dengan banyak halaman

3. Eksternal CSS

Eksternal CSS adalah metode yang paling direkomendasikan untuk menghubungkan CSS ke HTML. Gaya ditulis dalam file terpisah dengan ekstensi .css dan dihubungkan ke HTML menggunakan tag <link> di dalam <head>.

Struktur File:

📁 website/
📄 index.html
📁 css/
📄 style.css
📁 images/

File HTML (index.html):

HTML 
<!DOCTYPE html>
<html lang="id">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Website dengan Eksternal CSS</title>
    <link rel="stylesheet" href="css/style.css" />
    <!-- Menghubungkan file CSS eksternal -->
  </head>
  <body>
    <header class="site-header">
      <div class="container">
        <h1 class="logo">Nama Situs</h1>
        <nav class="main-nav">
          <ul>
            <li><a href="#" class="active">Beranda</a></li>
            <li><a href="#">Tentang</a></li>
            <li><a href="#">Layanan</a></li>
            <li><a href="#">Portofolio</a></li>
            <li><a href="#">Kontak</a></li>
          </ul>
        </nav>
      </div>
    </header>

    <main class="main-content">
      <section class="hero">
        <div class="container">
          <h2>Selamat Datang di Website Kami</h2>
          <p>Kami menyediakan solusi digital kreatif untuk bisnis Anda.</p>
          <a href="#" class="btn btn-primary">Mulai Sekarang</a>
        </div>
      </section>

      <section class="features">
        <div class="container">
          <h2 class="section-title">Mengapa Memilih Kami?</h2>
          <div class="features-grid">
            <div class="feature">
              <div class="feature-icon">🚀</div>
              <h3>Cepat</h3>
              <p>Waktu loading yang sangat cepat untuk pengalaman pengguna yang lebih baik.</p>
            </div>
            <div class="feature">
              <div class="feature-icon">🎨</div>
              <h3>Kreatif</h3>

              <h3>Responsif</h3>
              <p>Tampilan yang sempurna di semua perangkat.</p>

          </div>

      <div class="container">
        <p>© 2025 Nama Perusahaan. Semua hak dilindungi.</p>
      </div>
    </footer>
  </body>
</html>

Kelebihan Eksternal CSS:

  • Dapat digunakan di banyak halaman sekaligus
  • Membuat kode lebih bersih dan terorganisir
  • Meningkatkan kecepatan loading halaman (browser meng-cache file CSS)
  • Mudah untuk pemeliharaan dan pembaruan
  • Mengurangi duplikasi kode

Kekurangan Eksternal CSS:

  • Membutuhkan permintaan HTTP tambahan (tetapi diimbangi dengan caching)
  • Jika file CSS gagal dimuat, halaman akan kehilangan gaya

Kesimpulan

Memilih metode yang tepat untuk menghubungkan CSS ke HTML tergantung pada kebutuhan proyek Anda:

  • Inline CSS: Gunakan hanya untuk pengujian cepat atau untuk mengesampingkan gaya tertentu.
  • Internal CSS: Cocok untuk halaman tunggal atau ketika Anda perlu gaya yang hanya berlaku untuk satu halaman tertentu.
  • Eksternal CSS: Pilihan terbaik untuk sebagian besar proyek, terutama yang terdiri dari banyak halaman.

Untuk proyek yang lebih besar, Anda bahkan dapat membagi CSS Anda menjadi beberapa file eksternal (seperti reset.css, layout.css, components.css, dll.) untuk pengelolaan yang lebih baik.
