Atribut Penting dalam Formulir HTML

Memahami Action, Method, dan Name

Formulir HTML adalah komponen penting dalam pengembangan web yang memungkinkan pengguna untuk mengirim data ke server. Ada tiga atribut penting yang perlu dipahami dalam pembuatan formulir: action, method, dan name.

Catatan: Ketiga atribut ini bekerja sama untuk menentukan bagaimana data formulir akan diproses dan dikirim ke server.

1. Atribut Action

Atribut menentukan ke mana data formulir akan dikirim ketika formulir disubmit.

Contoh Penggunaan:

HTML 
<form action="/proses-pendaftaran.php">
  <!-- Isi formulir -->
</form>
Penjelasan:
  • Nilai dari action adalah URL yang akan memproses data formulir
  • Bisa berupa URL absolut (contoh: https://contoh.com/proses.php) atau relatif (contoh: /proses.php)
  • Jika tidak ditentukan, formulir akan mengirim data ke URL halaman saat ini

2. Atribut method

Atribut method menentukan metode HTTP yang akan digunakan untuk mengirim data formulir ke server.

Nilai yang Tersedia:

Nilai Keterangan
GET
  • Mengirim data melalui URL
  • Data terlihat di address bar browserPanjang URL terbatas (sekitar 2000 karakter)
  • Cocok untuk pencarian dan filter
POST
  • Mengirim data melalui body HTTP request
  • Tidak terlihat di address bar
  • Tidak ada batasan ukuran data
  • Cocok untuk data sensitif dan form pendaftaran

Contoh Perbandingan:

HTML 
<!-- Menggunakan GET -->
<form action="/cari" method="get">
  <input type="text" name="q" placeholder="Cari...">
  <button type="submit">Cari</button>
</form>

<!-- Menggunakan POST -->
<form action="/daftar" method="post">
  <input type="text" name="username" placeholder="Username">
  <input type="password" name="password" placeholder="Password">
  <button type="submit">Daftar</button>
</form>

3. Atribut name

Atribut name digunakan untuk mengidentifikasi data yang dikirim dari formulir.

Pentingnya Atribut Name:

  • Wajib untuk mengirim data formulir ke server
  • Nilai dari atribut name akan menjadi kunci (key) dari data yang dikirim
  • Berguna untuk mengakses nilai input di sisi server

Contoh Penggunaan:

HTML 
<form action="/proses.php" method="post">
  <div>
    <label for="nama">Nama Lengkap:</label>
    <input type="text" id="nama" name="nama_lengkap" required>
  </div>
  
  <div>
    <label for="email">Email:</label>
    <input type="email" id="email" name="email_pengguna" required>
  </div>
  
  <div>
    <label>Jenis Kelamin:</label>
    <input type="radio" id="laki" name="jenis_kelamin" value="L">
    <label for="laki">Laki-laki</label>
    
    <input type="radio" id="perempuan" name="jenis_kelamin" value="P">
    <label for="perempuan">Perempuan</label>
  </div>
  
  <div>
    <label for="hobi">Hobi:</label>
    <select id="hobi" name="hobi[]" multiple>
      <option value="membaca">Membaca</option>
      <option value="menulis">Menulis</option>
      <option value="berenang">Berenang</option>
    </select>
  </div>
  
  <button type="submit">Kirim</button>
</form>

Contoh Data yang Dikirim:

Data yang Dikirim 
// Data yang dikirim ke server
{
  "nama_lengkap": "Budi Santoso",
  "email_pengguna": "budi@contoh.com",
  "jenis_kelamin": "L",
  "hobi": ["membaca", "berenang"]
}

Contoh Formulir Lengkap

Berikut contoh formulir lengkap yang menggunakan ketiga atribut penting tersebut:

Formulir Pendaftaran Anggota


Penting! Selalu gunakan atribut name pada elemen formulir yang nilainya ingin Anda kirim ke server. Tanpa atribut name, nilai input tidak akan dikirim.

Kesimpulan

  • action: Menentukan ke mana data formulir akan dikirim
  • method: Menentukan cara pengiriman data (GET/POST)
  • name: Mengidentifikasi data yang dikirim ke server

Dengan memahami ketiga atribut penting ini, Anda dapat membuat formulir HTML yang efektif dan berfungsi dengan baik.
