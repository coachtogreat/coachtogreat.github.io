Pseudo-class CSS: :hover, :nth-child, dan Lainnya

Pseudo-class adalah kata kunci yang ditambahkan ke selector yang menentukan state khusus dari elemen yang dipilih. Mereka memungkinkan Anda untuk menerapkan gaya pada elemen tidak hanya berdasarkan isi dokumen, tetapi juga berdasarkan faktor eksternal seperti history navigasi, status konten, atau interaksi pengguna.

1. Pseudo-class Interaksi Pengguna

:hover

Mengganti gaya saat pengguna mengarahkan kursor ke atas elemen.

CSS 
/* Mengubah warna teks saat dihover */
a:hover {
  color: #e74c3c;
  text-decoration: underline;
}

/* Efek tombol saat dihover */
.btn:hover {
  background-color: #2980b9;
  transform: translateY(-2px);
  box-shadow: 0 4px 8px rgba(0,0,0,0.1);
}
Contoh :hover:
Arahkan kursor ke sini

:active

Mengganti gaya saat elemen sedang aktif (misalnya, saat tombol diklik).

CSS 
button:active {
  transform: translateY(1px);
  box-shadow: 0 1px 2px rgba(0,0,0,0.1);
}

:focus

Mengganti gaya saat elemen mendapatkan fokus (misalnya, input form).

CSS 
input:focus, textarea:focus, select:focus {
  border-color: #3498db;
  box-shadow: 0 0 0 3px rgba(52, 152, 219, 0.25);
  outline: none;
}
Contoh :focus:

2. Pseudo-class Berdasarkan Posisi

:first-child dan :last-child

Memilih elemen pertama/terakhir di antara saudara kandungnya.

CSS 
/* Elemen pertama dalam kontainer */
li:first-child {
  font-weight: bold;
  color: #e91e63;
}

/* Elemen terakhir dalam kontainer */
li:last-child {
  border-bottom: 2px solid #3f51b5;
  padding-bottom: 5px;
}

:nth-child()

Memilih elemen berdasarkan posisinya di antara saudara kandungnya.

CSS 
/* Setiap elemen ganjil */
tr:nth-child(odd) {
  background-color: #f8f9fa;
}

/* Setiap elemen genap */
tr:nth-child(even) {
  background-color: #e9ecef;
}

/* Elemen ke-3 */
li:nth-child(3) {
  color: #e91e63;
}

/* Setiap elemen kelipatan 3 */
li:nth-child(3n) {
  font-weight: bold;
}

/* 3 elemen pertama */
li:nth-child(-n+3) {
  border-left: 3px solid #4caf50;
}
Contoh :nth-child:
Item 1 (ganjil)
Item 2 (genap)
Item 3 (ganjil)
Item 4 (genap)
Item 5 (ganjil)

:nth-of-type()

Mirip dengan :nth-child, tetapi hanya menghitung elemen dengan tipe yang sama.

CSS 
/* Paragraf kedua dalam kontainer */
p:nth-of-type(2) {
  font-style: italic;
  color: #6c757d;
}

/* Gambar pertama dalam artikel */
article img:first-of-type {
  border: 3px solid #3498db;
  border-radius: 4px;
}

3. Pseudo-class Formulir

:checked

Memilih input checkbox atau radio yang dicentang.

/* Mengganti gaya checkbox yang dicentang */

  color: #28a745;
  font-weight: bold;
}

/* Menyembunyikan checkbox asli dan membuat yang kustom */
.custom-checkbox {

  height: 16px;
  margin-right: 8px;

  border-radius: 3px;
  vertical-align: middle;
}

  border-color: #28a745;
  content: "âœ“";

  text-align: center;

:disabled dan :enabled

Memilih input yang dinonaktifkan atau diaktifkan.

CSS 
input:disabled {
  background-color: #e9ecef;
  cursor: not-allowed;
}

input:enabled {
  background-color: #ffffff;
  border: 1px solid #ced4da;
}

:valid dan :invalid

Memilih input yang valid atau tidak valid berdasarkan validasi HTML5.

CSS 
input:valid {
  border-color: #28a745;
  background-image: url('data:image/svg+xml,');
  background-repeat: no-repeat;
  background-position: right calc(0.375em + 0.1875rem) center;
  background-size: calc(0.75em + 0.375rem) calc(0.75em + 0.375rem);
  padding-right: 2.25rem;
}

input:invalid {
  border-color: #dc3545;
  padding-right: 2.25rem;
  background-image: url('data:image/svg+xml,');
  background-repeat: no-repeat;
  background-position: right calc(0.375em + 0.1875rem) center;
  background-size: calc(0.75em + 0.375rem) calc(0.75em + 0.375rem);
}

4. Pseudo-class Lainnya

:not()

Menyingkirkan elemen tertentu dari seleksi.

CSS 
/* Semua paragraf kecuali yang memiliki class 'exclude' */
p:not(.exclude) {
  color: #333;
}

/* Semua tombol kecuali yang disabled */
button:not(:disabled) {
  cursor: pointer;
}

/* Semua elemen kecuali heading */
:not(h1):not(h2):not(h3):not(h4):not(h5):not(h6) {
  font-family: 'Arial', sans-serif;
}
Contoh :not():
Item 1 (biasa)
Item 2 (dikecualikan)
Item 3 (biasa)

:target

Memilih elemen yang menjadi target URL dengan identifier fragmen (contoh: #section1).

HTML 
<nav>
  <a href="#section1">Bagian 1</a>
  <a href="#section2">Bagian 2</a>
  <a href="#section3">Bagian 3</a>
</nav>

<section id="section1">...</section>
<section id="section2">...</section>
<section id="section3">...</section>
CSS 
section {
  padding: 2rem;
  margin: 1rem 0;
  border: 1px solid #ddd;
  border-radius: 4px;
  transition: all 0.3s ease;
}

section:target {
  background-color: #e7f5ff;
  border-color: #4dabf7;
  box-shadow: 0 0 0 0.2rem rgba(13, 110, 253, 0.25);
  animation: highlight 1.5s ease;
}

@keyframes highlight {
  from { background-color: #fff3cd; }
  to { background-color: #e7f5ff; }
}

:lang()

Memilih elemen berdasarkan kode bahasa yang ditentukan.

CSS 
/* Gaya untuk konten berbahasa Indonesia */
:lang(id) {
  font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
}

/* Gaya untuk kutipan dalam bahasa Inggris */
q:lang(en) {
  quotes: '\201C' '\201D' '\2018' '\2019';
  font-style: italic;
  color: #495057;
}

Ringkasan Pseudo-class Umum:

Pseudo-class Contoh Deskripsi
:hover a:hover Ketika kursor berada di atas elemen
:active button:active Ketika elemen sedang diaktifkan (diklik)
:focus input:focus Ketika elemen mendapatkan fokus
:first-child li:first-child Elemen pertama di antara saudara kandungnya
:nth-child() tr:nth-child(odd) Elemen berdasarkan posisi (ganjil, genap, atau rumus)
:checked input:checked Checkbox atau radio yang dicentang
:disabled button:disabled Elemen input yang dinonaktifkan
:not() div:not(.exclude) Menyingkirkan elemen dari seleksi
:target #section1:target Elemen yang menjadi target URL dengan identifier fragmen
:lang() :lang(id) Elemen dengan atribut bahasa tertentu
