Glosarium Istilah

A

Aksesibilitas

Kemampuan semua orang untuk menggunakan, memahami, dan berinteraksi dengan konten digital tanpa hambatan.

AI (Artificial Intelligence)

Teknologi yang memungkinkan komputer untuk melakukan tugas-tugas yang biasanya membutuhkan kecerdasan manusia.

API (Application Programming Interface)

Set aturan dan protokol yang memungkinkan perangkat lunak berbeda untuk berkomunikasi satu sama lain.

B

Blockchain

Teknologi buku besar digital yang mencatat transaksi secara aman dan tidak dapat diubah.

Browser

Program komputer yang digunakan untuk menampilkan halaman web dan mengakses internet.

C

ChatGPT

Model bahasa AI yang dikembangkan oleh OpenAI untuk percakapan dan penulisan konten.

Content Editor

Alat yang digunakan untuk membuat dan mengedit konten digital.

D

Disabilitas

Kondisi yang membatasi kemampuan seseorang untuk melakukan aktivitas tertentu.

DOM (Document Object Model)

Representasi struktural dari dokumen web yang dapat dimanipulasi oleh JavaScript.

E

Editor

Perangkat lunak yang digunakan untuk membuat dan mengedit teks atau konten.

EN 301 549

Standar Eropa untuk aksesibilitas produk dan layanan digital.

F

Format

Struktur atau tata letak konten digital.

Functionality

Kemampuan sistem untuk melakukan tugas-tugas tertentu.

G

GitHub

Platform untuk kolaborasi pengembangan perangkat lunak.

GPT (Generative Pre-trained Transformer)

Model AI yang dapat menghasilkan teks berdasarkan input yang diberikan.

H

HTML (HyperText Markup Language)

Bahasa markup yang digunakan untuk membuat halaman web.

HTTP (HyperText Transfer Protocol)

Protokol yang digunakan untuk mentransfer data di internet.

I

Inklusivitas

Prinsip yang menjamin semua orang dapat mengakses dan menggunakan konten digital.

Interface

Antarmuka yang memungkinkan interaksi antara pengguna dan sistem.

J

JavaScript

Bahasa pemrograman yang digunakan untuk membuat interaksi di halaman web.

JAWS (Job Access With Speech)

Screen reader yang membantu pengguna tunanetra mengakses konten digital.

K

Keyboard Navigation

Kemampuan untuk menavigasi antarmuka menggunakan keyboard saja.

Konten

Informasi atau data yang ditampilkan di halaman web.

L

Layout

Susunan atau tata letak elemen-elemen di halaman web.

Level

Tingkat kepatuhan atau kualitas dalam konteks aksesibilitas.

M

Metadata

Data yang menggambarkan informasi tentang konten lainnya.

Mobile

Perangkat atau aplikasi yang dapat digunakan di perangkat mobile.

N

Navigation

Proses menemukan dan berpindah antara halaman atau elemen web.

NVDA (NonVisual Desktop Access)

Screen reader gratis yang membantu pengguna tunanetra mengakses konten digital.

O

Optimasi

Proses memperbaiki kinerja atau kualitas sistem.

Output

Hasil atau produk yang dihasilkan oleh sistem.

P

Performance

Kinerja atau efisiensi sistem dalam melakukan tugas-tugasnya.

Plugin

Komponen tambahan yang dapat ditambahkan ke sistem untuk menambah fungsionalitas.

Q

Quality

Tingkat keunggulan atau kualitas dari suatu produk atau layanan.

Query

Pertanyaan atau permintaan yang diajukan ke sistem.

R

Regulasi

Aturan atau peraturan yang mengatur penggunaan teknologi.

Reporting

Proses membuat dan menyajikan laporan atau informasi.

S

Semantic

Struktur dan makna dari konten yang dapat dipahami oleh mesin.

Screen Reader

Perangkat lunak yang membaca teks di layar untuk pengguna tunanetra.

T

Technology

Alat atau metode yang digunakan untuk mencapai tujuan tertentu.

Testing

Proses memeriksa kualitas atau kinerja sistem.

U

UI (User Interface)

Antarmuka yang memungkinkan pengguna berinteraksi dengan sistem.

UX (User Experience)

Pengalaman pengguna dalam menggunakan sistem atau produk.

V

Validation

Proses memeriksa kebenaran atau kevalidan data atau informasi.

VoiceOver

Screen reader yang dikembangkan oleh Apple untuk pengguna tunanetra.

W

Web

Jaringan global yang menghubungkan halaman web dan situs web.

W3C (World Wide Web Consortium)

Organisasi internasional yang mengembangkan standar web.

X

XHTML

Versi XML dari HTML yang lebih ketat dalam sintaksnya.

XML (eXtensible Markup Language)

Bahasa markup yang dapat diperluas untuk menyimpan dan mentransfer data.
