Membuat Halaman Artikel dengan Gambar dan Link

Panduan Lengkap untuk Pemula

Halaman artikel yang baik membutuhkan lebih dari sekadar teks. Gambar dan link adalah elemen penting yang membuat konten lebih menarik dan informatif. Mari pelajari cara membuat halaman artikel yang profesional dengan HTML dan CSS.

1. Struktur Dasar Artikel

HTML 
<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Judul Artikel</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <article class="article-container">
    <header class="article-header">
      <h1>Judul Artikel yang Menarik</h1>
      <div class="article-meta">
        <span class="author">Oleh: Nama Penulis</span>
        <span class="date">14 Mei 2025</span>
      </div>
    </header>
    
    <div class="article-content">
      <!-- Konten artikel akan ditambahkan di sini -->
    </div>
    
    <footer class="article-footer">
      <div class="tags">
        <span>Tag:</span>
        <a href="#">HTML</a>
        <a href="#">CSS</a>
        <a href="#">Web Design</a>
      </div>
    </footer>
  </article>
</body>
</html>

2. Menambahkan Gambar ke Dalam Artikel

HTML 
<figure class="article-image">
  <img 
    src="gambar-contoh.jpg" 
    alt="Deskripsi gambar yang informatif"
    loading="lazy"
  >
  <figcaption>
    Gambar 1: Contoh gambar dengan caption yang deskriptif
  </figcaption>
</figure>

3. Menambahkan Link ke Dalam Teks

HTML 
<p>
  HTML adalah bahasa markup standar untuk membuat halaman web. 
  Anda dapat mempelajari lebih lanjut tentang 
  <a href="https://developer.mozilla.org/id/docs/Web/HTML" 
     target="_blank" 
     rel="noopener noreferrer"
  >
    HTML di MDN Web Docs
  </a>.
</p>

<p>
  Untuk informasi lebih lanjut, kunjungi 
  <a href="#" class="read-more">halaman ini</a>.
</p>

6. Menambahkan Artikel Contoh

HTML 
<article class="article-container">
  <header class="article-header">
    <h1>Panduan Lengkap Belajar HTML dan CSS</h1>
    <div class="article-meta">
      <span class="author">Oleh: Web Developer</span>
      <span class="date">14 Mei 2025</span>
    </div>
  </header>
  
  <div class="article-content">
    <p>
      HTML (HyperText Markup Language) dan CSS (Cascading Style Sheets) 
      adalah dua teknologi dasar untuk membangun halaman web. 
      <a href="https://developer.mozilla.org/id/docs/Web/HTML" 
         target="_blank" 
         rel="noopener noreferrer"
      >
        HTML
      </a> 
      digunakan untuk struktur konten, sementara 
      <a href="https://developer.mozilla.org/id/docs/Web/CSS" 
         target="_blank" 
         rel="noopener noreferrer"
      >
        CSS
      </a> 
      digunakan untuk gaya dan tata letak.
    </p>
    
    <figure class="article-image">
      <img 
        src="html-css.jpg" 
        alt="Ilustrasi HTML dan CSS"
        loading="lazy"
      >
      <figcaption>
        Gambar: Ilustrasi HTML dan CSS bekerja sama dalam pembuatan web
      </figcaption>
    </figure>
    
    <h2>Keuntungan Menggunakan HTML dan CSS</h2>
    
    <p>
      Berikut adalah beberapa keuntungan menggunakan HTML dan CSS:
    </p>
    
    <ul>
      <li>Mudah dipelajari untuk pemula</li>
      <li>Didukung oleh semua browser web modern</li>
      <li>Ringan dan cepat dimuat</li>
      <li>Dapat diakses oleh perangkat apa pun</li>
    </ul>
    
    <h2>Struktur Dasar HTML</h2>
    
    <p>
      Setiap dokumen HTML dimulai dengan deklarasi <code>&lt;!DOCTYPE html&gt;</code> 
      dan memiliki elemen <code>&lt;html&gt;</code> yang berisi <code>&lt;head&gt;</code> 
      dan <code>&lt;body&gt;</code>.
    </p>
    
    <h2>Kesimpulan</h2>
    
    <p>
      HTML dan CSS adalah fondasi dari pengembangan web modern. 
      Dengan mempelajari keduanya, Anda dapat membuat berbagai jenis 
      halaman web yang menarik dan fungsional.
    </p>
  </div>
  
  <footer class="article-footer">
    <div class="tags">
      <span>Tag:</span>
      <a href="#">HTML</a>
      <a href="#">CSS</a>
      <a href="#">Web Development</a>
      <a href="#">Pemrograman Web</a>
    </div>
  </footer>
</article>
Tips untuk Artikel yang Lebih Baik:
  • Gunakan gambar yang relevan dan berkualitas tinggi
  • Optimalkan ukuran gambar untuk web
  • Gunakan heading dengan hierarki yang benar (h1, h2, h3, dll.)
  • Tambahkan alt text untuk semua gambar
  • Gunakan link yang deskriptif, bukan "klik di sini"
  • Pastikan konten mudah dibaca dengan font dan ukuran yang sesuai
