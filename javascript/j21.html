Menulis Kode JavaScript Pertama

1. Hallo Dunia dengan JavaScript

Contoh Pertama: Hello World 
// Cara 1: Menggunakan console.log
console.log("Halo, Dunia!");

// Cara 2: Menggunakan alert
alert("Halo, Dunia!");

// Cara 3: Mengubah isi halaman
let paragraf = document.createElement("p");
paragraf.textContent = "Halo, Dunia!";
document.body.appendChild(paragraf);

2. Variabel dan Tipe Data Dasar

Contoh: Menggunakan Variabel 
// Deklarasi variabel
let nama = "John";
let umur = 25;
let isStudent = true;

// Menampilkan variabel
console.log("Nama: " + nama);
console.log("Umur: " + umur);
console.log("Status: " + isStudent);

3. Operasi Matematika Sederhana

Contoh: Operasi Matematika 
// Operasi penjumlahan
let a = 5;
let b = 3;
let hasil = a + b;
console.log("Hasil: " + hasil);

// Operasi pengurangan
let c = 10;
let d = 4;
let hasilKurang = c - d;
console.log("Hasil: " + hasilKurang);

4. Menggunakan Event

Contoh: Menggunakan Event 
<!-- HTML -->
<button id="tombol">Klik Saya</button>

<!-- JavaScript -->
<script>
document.getElementById("tombol").addEventListener("click", function() {
  alert("Tombol diklik!");
});
</script>

5. Mengubah Konten Halaman

Contoh: Mengubah Konten 
<!-- HTML -->
<p id="teks">Ini adalah teks asli.</p>

<!-- JavaScript -->
<script>
// Mengubah teks
let paragraf = document.getElementById("teks");
paragraf.textContent = "Teks ini telah berubah!";

// Mengubah warna
paragraf.style.color = "blue";
</script>

6. Tips untuk Pemula

  • Mulai dengan contoh sederhana
  • Gunakan console.log untuk debugging
  • Praktekkan kode di browser console
  • Baca pesan error dengan seksama
  • Gunakan komentar untuk menjelaskan kode

7. Latihan Praktis

  1. Buat program yang menampilkan nama Anda
  2. Buat kalkulator sederhana
  3. Buat tombol yang mengubah warna halaman
  4. Buat program yang menghitung umur dari tahun lahir
  5. Buat form sederhana dengan validasi
