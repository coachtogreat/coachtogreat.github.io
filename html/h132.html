Memahami Box Model

Panduan Lengkap tentang Box Model

Box Model adalah konsep fundamental yang menjelaskan bagaimana elemen HTML ditampilkan di halaman web. Setiap elemen HTML dapat dianggap sebagai kotak yang terdiri dari konten, padding, border, dan margin.

1. Komponen Box Model

Box Model terdiri dari empat komponen utama:

Konten
Penjelasan Komponen:
  • Konten (Content) - Area yang berisi teks, gambar, atau elemen lainnya
  • Padding - Ruang antara konten dan border
  • Border - Garis yang mengelilingi padding dan konten
  • Margin - Ruang di luar border yang memisahkan elemen dengan elemen lain

2. Ukuran Kotak (Box Sizing)

Secara default, lebar dan tinggi elemen hanya berlaku untuk konten. Padding dan border ditambahkan ke ukuran tersebut.

CSS 
.box {
  width: 300px;
  height: 100px;
  padding: 20px;
  border: 10px solid #0d6efd;
  margin: 30px;
  background-color: #e9ecef;
}

Total lebar elemen di atas adalah:

lebar total = margin-kiri + border-kiri + padding-kiri + width + padding-kanan + border-kanan + margin-kanan

lebar total = 30px + 10px + 20px + 300px + 20px + 10px + 30px = 420px

Perhatian: Perilaku default ini bisa menyebabkan masalah tata letak yang tidak terduga. Untuk menghindarinya, gunakan box-sizing: border-box.

3. Menggunakan box-sizing: border-box

Dengan box-sizing: border-box, padding dan border termasuk dalam lebar dan tinggi yang ditentukan.

CSS 
/* Menerapkan ke semua elemen */
* {
  box-sizing: border-box;
  margin: 0;
  padding: 0;
}

.box {
  width: 300px;  /* Termasuk padding dan border */
  padding: 20px;
  border: 10px solid #0d6efd;
}
Tips Praktis:
  • Selalu gunakan box-sizing: border-box di awal file CSS Anda
  • Gunakan unit relatif seperti rem atau % untuk ukuran yang lebih responsif
  • Gunakan shorthand properties untuk menulis kode yang lebih ringkas

4. Margin Collapsing

Margin atas dan bawah dari dua elemen yang berdekatan bisa saling tumpang tindih (collapse).

HTML 
<div class="box" style="margin-bottom: 30px;">Kotak 1</div>
<div class="box" style="margin-top: 40px;">Kotak 2</div>

Jarak antar kotak di atas adalah 40px (bukan 70px) karena margin collapsing.

Kapan Margin Collapsing Terjadi:
  • Antara elemen blok yang berdekatan
  • Antara parent dan child pertama/terakhir (jika parent tidak memiliki border, padding, atau konten inline)
  • Pada elemen kosong (tidak memiliki border, padding, atau tinggi yang ditentukan)

5. Contoh Praktis

Berikut contoh penerapan box model dalam membuat card sederhana:

HTML 
<div class="card">
  <img src="gambar.jpg" alt="Contoh Gambar" class="card-img">
  <div class="card-body">
    <h3 class="card-title">Judul Card</h3>
    <p class="card-text">Ini adalah contoh card sederhana dengan box model.</p>
    <a href="#" class="btn">Baca Selengkapnya</a>
  </div>
</div>
CSS 
/* Reset dan Box Sizing */
* {
  box-sizing: border-box;
  margin: 0;
  padding: 0;
}

/* Style Card */
.card {
  width: 300px;
  border: 1px solid #dee2e6;
  border-radius: 8px;
  overflow: hidden;
  margin: 20px;
  box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
}

.card-img {
  width: 100%;
  height: 200px;
  object-fit: cover;
}

.card-body {
  padding: 20px;
}

.card-title {
  margin-bottom: 10px;
  color: #333;
}

.card-text {
  color: #6c757d;
  margin-bottom: 15px;
  line-height: 1.5;
}

.btn {
  display: inline-block;
  background-color: #0d6efd;
  color: white;
  text-decoration: none;
  padding: 8px 16px;
  border-radius: 4px;
  font-size: 14px;
  transition: background-color 0.3s;
}

.btn:hover {
  background-color: #0b5ed7;
}
Best Practices Box Model:
  • Selalu gunakan box-sizing: border-box untuk perhitungan ukuran yang lebih mudah
  • Gunakan margin untuk jarak antar elemen
  • Gunakan padding untuk jarak antara border dan konten
  • Manfaatkan shorthand properties (contoh: margin: 10px 20px;)
  • Gunakan unit relatif seperti rem atau em untuk ukuran yang responsif
