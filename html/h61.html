Struktur Dasar Tabel HTML

Memahami Elemen Tabel dalam HTML

Tabel dalam HTML digunakan untuk menampilkan data dalam bentuk baris dan kolom. Tabel terdiri dari sel-sel yang disusun dalam baris dan kolom, dengan header untuk mengidentifikasi data.

1. Elemen Dasar Tabel

Berikut adalah elemen-elemen dasar yang digunakan dalam membuat tabel HTML:

HTML 
<table>
  <caption>Judul Tabel</caption>
  <thead>
    <tr>
      <th>Header 1</th>
      <th>Header 2</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Data 1</td>
      <td>Data 2</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td>Footer 1</td>
      <td>Footer 2</td>
    </tr>
  </tfoot>
</table>
Contoh Tabel Sederhana
Nama Usia Kota
Andi 25 Jakarta
Budi 30 Bandung
Jumlah Data 2

2. Tag Dasar Tabel

Tag Deskripsi Contoh
<table> Membungkus seluruh elemen tabel <table>...</table>
<caption> Judul atau deskripsi tabel <caption>Daftar Mahasiswa</caption>
<thead> Mengelompokkan header tabel <thead>...</thead>
<tbody> Mengelompokkan isi tabel <tbody>...</tbody>
<tfoot> Mengelompokkan footer tabel <tfoot>...</tfoot>
<tr> Baris dalam tabel <tr>...</tr>
<th> Sel header (teks tebal dan rata tengah) <th>Nama</th>
<td> Sel data <td>Andi</td>

3. Menggabungkan Sel

Anda dapat menggabungkan sel menggunakan atribut colspan dan rowspan:

Menggunakan colspan (menggabungkan kolom)
HTML 
<table border="1">
  <tr>
    <th colspan="2">Nama Lengkap</th>
    <th>Usia</th>
  </tr>
  <tr>
    <td>Nama</td>
    <td>Nama Belakang</td>
    <td>Umur</td>
  </tr>
</table>
Nama Lengkap Usia
Nama Nama Belakang Umur
Menggunakan rowspan (menggabungkan baris)
HTML 
<table border="1">
  <tr>
    <th rowspan="2">No.</th>
    <th colspan="2">Nama</th>
  </tr>
  <tr>
    <th>Depan</th>
    <th>Belakang</th>
  </tr>
  <tr>
    <td>1</td>
    <td>Andi</td>
    <td>Susanto</td>
  </tr>
</table>
No. Nama
Depan Belakang
1 Andi Susanto
2 Budi Santoso

4. Tabel dengan CSS Dasar

Berikut contoh penerapan CSS dasar untuk mempercantik tampilan tabel:

HTML & CSS 
<style>

    border-collapse: collapse;
    margin: 25px 0;
    font-size: 0.9em;
    font-family: sans-serif;

    box-shadow: 0 0 20px rgba(0, 0, 0, 0.15);
  }
  .tabel-rapi thead tr {
    background-color: #009879;
    color: #ffffff;
    text-align: left;
  }
  .tabel-rapi th,
  .tabel-rapi td {
    padding: 12px 15px;
  }
  .tabel-rapi tbody tr {
    border-bottom: 1px solid #dddddd;
  }
  .tabel-rapi tbody tr:nth-of-type(even) {
    background-color: #f3f3f3;
  }
  .tabel-rapi tbody tr:last-of-type {
    border-bottom: 2px solid #009879;
  }
  .tabel-rapi tbody tr:hover {
    font-weight: bold;
    color: #009879;
  }
</style>

5. Praktik Terbaik

  • Selalu gunakan <thead>, <tbody>, dan <tfoot> untuk struktur yang lebih baik
  • Gunakan <th> untuk header kolom atau baris
  • Tambahkan <caption> untuk deskripsi tabel
  • Gunakan scope pada <th> untuk aksesibilitas
  • Hindari tabel bersarang yang terlalu dalam
  • Gunakan CSS untuk styling, bukan atribut HTML yang sudah usang seperti border, cellpadding, dll.
  • Pastikan tabel responsif di perangkat mobile
  • Gunakan colspan dan rowspan dengan bijak

6. Contoh Tabel Kompleks

Berikut contoh tabel dengan berbagai fitur yang telah dipelajari:

HTML 
<table class="tabel-kompleks">
  <caption>Jadwal Pelajaran Kelas 10A</caption>
  <colgroup>
    <col span="1" style="background-color:#f1f1f1">
    <col span="5" style="background-color:#ffffff">
  </colgroup>
  <thead>
    <tr>
      <th scope="col">Jam</th>
      <th scope="col">Senin</th>
      <th scope="col">Selasa</th>
      <th scope="col">Rabu</th>
      <th scope="col">Kamis</th>
      <th scope="col">Jumat</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th scope="row">07.30 - 09.00</th>
      <td>Matematika</td>
      <td>Bahasa Inggris</td>
      <td rowspan="2">Fisika</td>
      <td>Kimia</td>
      <td>Sejarah</td>
    </tr>
    <tr>
      <th scope="row">09.00 - 10.30</th>
      <td>Bahasa Indonesia</td>
      <td>Biologi</td>
      <td colspan="2">Pendidikan Jasmani</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td colspan="6">* Jadwal dapat berubah sewaktu-waktu</td>
    </tr>
  </tfoot>
</table>
Jadwal Pelajaran Kelas 10A
Jam Senin Selasa Rabu Kamis Jumat
07.30 - 09.00 Matematika Bahasa Inggris Fisika Kimia Sejarah
09.00 - 10.30 Bahasa Indonesia Biologi Pendidikan Jasmani
* Jadwal dapat berubah sewaktu-waktu
