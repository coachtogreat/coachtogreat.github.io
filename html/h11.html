Pengenalan HTML

Dasar-dasar HTML

HTML (HyperText Markup Language) adalah bahasa markup standar untuk membuat halaman web. Berikut adalah konsep dasar HTML:

1. Struktur Dasar HTML

HTML 
<!DOCTYPE html>
<html>
  <head>
    <title>Judul Halaman</title>
  </head>
  <body>
    <h1>Ini adalah Heading</h1>
    <p>Ini adalah paragraf.</p>
  </body>
</html>

2. Elemen Dasar HTML

  • <!DOCTYPE html> - Mendeklarasikan tipe dokumen HTML5
  • <html> - Elemen akar dari halaman HTML
  • <head> - Berisi metadata/informasi dokumen
  • <title> - Menentukan judul halaman (muncul di tab browser)
  • <body> - Berisi konten yang ditampilkan di halaman web

3. Tag HTML Umum

  • <h1> sampai <h6> - Heading/tingkat judul
  • <p> - Paragraf teks
  • <a href="url">Link</a> - Hyperlink
  • <img src="gambar.jpg" alt="deskripsi"> - Gambar
  • <ul> dan <ol> - Daftar tidak berurutan dan berurutan
  • <li> - Item dalam daftar
  • <div> - Kontainer blok untuk elemen lainnya
  • <span> - Kontainer inline untuk teks

4. Atribut Dasar

  • class="nama-kelas" - Menentukan nama class untuk CSS/JavaScript
  • id="identifikasi" - Menentukan ID unik untuk elemen
  • style="property:value;" - CSS inline
  • src="url" - Menentukan sumber (gambar, script, dll)
  • alt="teks" - Teks alternatif untuk aksesibilitas

5. Komentar HTML

HTML 
<!-- Ini adalah komentar HTML -->
<!-- Komentar tidak akan ditampilkan di browser -->

6. Atribut Dasar HTML

  • Semua elemen HTML dapat memiliki atribut
  • Atribut memberikan informasi tambahan tentang elemen
  • Atribut selalu ditulis di tag pembuka
  • Atribut biasanya dalam format name="value"

7. Karakter Khusus

Beberapa karakter khusus di HTML:

  • &lt; - < (less than)
  • &gt; - > (greater than)
  • &amp; - & (ampersand)
  • &quot; - " (quotation mark)
  • &nbsp; - Non-breaking space

Dengan memahami dasar-dasar ini, Anda sudah siap untuk mulai membuat halaman web sederhana menggunakan HTML.
