Selector Kombinasi CSS: Descendant, Child, dan Sibling

Selector kombinasi memungkinkan kita untuk memilih elemen berdasarkan hubungan hierarkisnya dengan elemen lain. Ini sangat berguna ketika kita ingin menargetkan elemen tertentu dalam struktur HTML yang kompleks.

1. Descendant Selector (Spasi)

Descendant selector memilih semua elemen yang merupakan turunan (anak, cucu, cicit, dst) dari elemen yang ditentukan.

Contoh:

CSS 
/* Memilih semua elemen 
 yang berada di dalam 
 */
div p {
  color: #2c3e50;
  line-height: 1.6;
}

/* Contoh lebih spesifik */
.container article p {
  margin-bottom: 1rem;
}
Contoh Penerapan:
Ini adalah parent
Child 1 (langsung)
Child 2 (cucu)

Paragraf di dalam child 2

Paragraf langsung di dalam parent

Perhatian: Semua elemen <p> di dalam .parent akan terkena style, tidak peduli seberapa dalam levelnya.

2. Child Selector (>)

Child selector memilih semua elemen yang merupakan anak langsung dari elemen yang ditentukan.

Contoh:

CSS 
/* Hanya memilih 
 yang merupakan anak langsung dari 
 */
div > p {
  color: #e74c3c;
  font-weight: bold;
}

/* Contoh dengan class */
.menu > li {
  display: inline-block;
  margin-right: 1rem;
}
Contoh Penerapan:
Ini adalah parent
Child 1 (langsung)

Paragraf di dalam child 1

Child 2 (cucu)

Paragraf di dalam child 2

Paragraf langsung di dalam parent

Perhatian: Hanya elemen <p> yang merupakan anak langsung dari .parent yang akan terkena style. Paragraf yang ada di dalam child tidak akan terpengaruh.

3. Adjacent Sibling Selector (+)

Adjacent sibling selector memilih elemen yang berada tepat setelah elemen yang ditentukan dan memiliki parent yang sama.

Contoh:

CSS 
/* Memilih 
 yang berada tepat setelah 
 */
h2 + p {
  margin-top: 0.5rem;
  color: #8e44ad;
}

/* Contoh dengan class */
.alert + p {
  margin-top: 1.5rem;
}
Contoh Penerapan:

Judul

Paragraf ini akan terkena style karena berada tepat setelah h3

Paragraf ini TIDAK akan terkena style

Div Sibling 1

Paragraf setelah sibling 1 (akan terkena style)

Div Sibling 2

Paragraf setelah sibling 2 (akan terkena style)

4. General Sibling Selector (~)

General sibling selector memilih semua elemen yang berada setelah elemen yang ditentukan dan memiliki parent yang sama.

Contoh:

CSS 
/* Memilih semua 
 yang berada setelah 
 */
h2 ~ p {
  color: #16a085;
  font-style: italic;
}

/* Contoh dengan class */
.section-title ~ p {
  margin-left: 1rem;
}
Contoh Penerapan:

Judul Bagian

Paragraf 1 setelah judul

Paragraf 2 setelah judul

Div Pembatas

Paragraf setelah div pembatas

Paragraf terakhir

Perhatian: Semua elemen <p> yang berada setelah <h3> akan terkena style, tidak peduli apakah ada elemen lain di antaranya.

Ringkasan Selector Kombinasi:

Selector Contoh Deskripsi
div p Descendant Memilih semua <p> di dalam <div>
div > p Child Memilih <p> yang merupakan anak langsung <div>
h2 + p Adjacent Sibling Memilih <p> yang berada tepat setelah <h2>
h2 ~ p General Sibling Memilih semua <p> yang berada setelah <h2>
