Pengenalan HTML

Struktur Dasar HTML

HTML (HyperText Markup Language) adalah bahasa markup standar untuk membuat halaman web. Setiap dokumen HTML memiliki struktur dasar yang terdiri dari beberapa elemen penting yang bekerja sama untuk menampilkan konten di browser. Memahami struktur ini sangat penting untuk pengembangan web yang baik dan benar.

Berikut adalah penjelasan mendalam tentang setiap bagian dari struktur dasar HTML:

HTML 
<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Judul Halaman</title>
</head>
<body>
  <h1>Halo, Dunia!</h1>
  <p>Ini adalah paragraf pertama.</p>
</body>
</html>

Penjelasan Mendalam Setiap Elemen

<!DOCTYPE html>

  • Deklarasi Tipe Dokumen yang harus ada di setiap halaman HTML.
  • Memberi tahu browser bahwa dokumen ini menggunakan standar HTML5.
  • Bukan sebuah tag HTML, melainkan instruksi untuk browser.
  • Harus diletakkan di baris pertama dokumen, sebelum tag <html>.
  • Tidak peka huruf besar/kecil, tetapi penulisan <!DOCTYPE html> direkomendasikan.

<html lang="id"> ... </html>

  • Elemen root yang membungkus seluruh konten halaman HTML.
  • Atribut lang="id" penting untuk:
    • Pembaca layar (screen readers)
    • Mesin pencari (SEO)
    • Koreksi ejaan otomatis
    • Terjemahan otomatis
  • Selalu tutup dengan tag </html> di akhir dokumen.

<head>...</head>

  • Kontainer untuk metadata (data tentang data) yang tidak ditampilkan di halaman web.
  • Berisi informasi penting untuk browser dan mesin pencari.
  • Elemen-elemen yang umumnya ada di dalam <head>:

<meta charset="UTF-8">

  • Mendefinisikan set karakter yang digunakan dalam dokumen.
  • UTF-8 mencakup hampir semua karakter dan simbol dari seluruh dunia.
  • Mencegah masalah dengan karakter khusus seperti é, ñ, 漢字, dll.
  • Harus diletakkan dalam 1024 byte pertama dokumen.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

  • Mengontrol tampilan halaman di perangkat mobile.
  • width=device-width: Menyesuaikan lebar halaman dengan lebar layar perangkat.
  • initial-scale=1.0: Menetapkan level zoom awal saat halaman pertama kali dimuat.
  • Parameter tambahan yang berguna:
    • maximum-scale=1.0: Mencegah zoom
    • user-scalable=no: Menonaktifkan zoom manual

<title>Judul Halaman</title>

  • Menentukan judul yang ditampilkan di tab browser dan hasil pencarian.
  • Penting untuk SEO (Search Engine Optimization).
  • Idealnya antara 50-60 karakter.
  • Harus deskriptif dan relevan dengan konten halaman.

<body>...</body>

  • Berisi semua konten yang akan ditampilkan kepada pengguna.
  • Elemen dasar yang umum digunakan di dalam <body>:
    • <header>: Bagian kepala halaman
    • <nav>: Navigasi
    • <main>: Konten utama
    • <section>: Bagian dari halaman
    • <article>: Konten mandiri
    • <footer>: Bagian kaki halaman
  • Hanya boleh ada satu elemen <body> dalam satu dokumen HTML.

Contoh isi:

  • <h1> sampai <h6>: heading (judul), di mana <h1> adalah yang paling penting.
  • <p>: paragraf teks.
  • <a>: tautan (link).
  • <img>: gambar.
  • <div> dan <span>: elemen pembungkus (container), digunakan untuk struktur dan styling.

Catatan Penting

  • HTML bersifat hierarkis (bertingkat), sehingga indentasi (spasi) penting untuk keterbacaan, meskipun tidak wajib.
  • HTML bersifat case-insensitive, tapi konvensi umum menulis tag dalam huruf kecil.
