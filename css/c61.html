Media Queries di CSS: Membuat Tampilan Adaptif

Media Queries adalah fitur CSS yang memungkinkan kita menerapkan gaya yang berbeda berdasarkan karakteristik perangkat, seperti lebar layar, orientasi, atau resolusi. Dengan Media Queries, kita bisa membuat desain web yang responsif dan terlihat bagus di berbagai perangkat.

Breakpoint Umum:

  • max-width: 600px: Perangkat mobile
  • min-width: 600px: Tablet dan lebih besar
  • min-width: 768px: Tablet landscape
  • min-width: 992px: Desktop
  • min-width: 1200px: Layar lebar

1. Sintaks Dasar Media Query

Media Query dimulai dengan @media diikuti dengan kondisi tertentu. Berikut adalah contoh sederhana:

CSS 
/* Gaya default */
body {
  background-color: white;
  color: black;
}

/* Gaya untuk layar dengan lebar maksimal 600px */
@media (max-width: 600px) {
  body {
    background-color: lightblue;
    color: darkblue;
  }
}

2. Contoh Media Query untuk Berbagai Perangkat

Berikut contoh penerapan Media Query untuk berbagai ukuran layar:

CSS 
/* Gaya default untuk mobile first */
.container {
  width: 100%;
  padding: 10px;
}

/* Tablet - 600px ke atas */
@media (min-width: 600px) {
  .container {
    width: 90%;
    margin: 0 auto;
    padding: 15px;
  }
}

/* Desktop - 992px ke atas */
@media (min-width: 992px) {
  .container {
    width: 80%;
    max-width: 1200px;
    padding: 20px;
  }
}

3. Demo Responsif

Coba ubah ukuran jendela browser Anda untuk melihat efek Media Query pada kotak di bawah ini:

Ukuran Layar:
CSS 
.responsive-demo {
  width: 100%;
  height: 100px;
  background-color: #e3f2fd;
  display: flex;
  align-items: center;
  justify-content: center;
  transition: all 0.3s ease;
}

/* Gaya untuk layar kecil */
@media (max-width: 600px) {
  .responsive-demo {
    background-color: #e8f5e9; /* Hijau muda */
  }
}

/* Gaya untuk layar besar */
@media (min-width: 1024px) {
  .responsive-demo {
    background-color: #fff3e0; /* Oranye muda */
  }
}

4. Media Query untuk Orientasi

Media Query juga bisa digunakan untuk menargetkan orientasi perangkat:

CSS 
/* Potret (tinggi > lebar) */
@media (orientation: portrait) {
  body {
    background-color: #fff8e1;
  }
}

/* Lanskap (lebar > tinggi) */
@media (orientation: landscape) {
  body {
    background-color: #e8f5e9;
  }
}

/* Kombinasi dengan lebar layar */
@media (max-width: 600px) and (orientation: landscape) {
  body {
    font-size: 14px;
  }
}

5. Media Query untuk Resolusi Tinggi (Retina Display)

Untuk perangkat dengan layar beresolusi tinggi seperti Retina Display:

CSS 
/* Untuk layar dengan kepadatan piksel 2x */
@media 
  (-webkit-min-device-pixel-ratio: 2), 
  (min-resolution: 192dpi) { 
  .logo {
    background-image: url('logo@2x.png');
    background-size: contain;
  }
}

Tips Penggunaan Media Query:

  • Gunakan pendekatan Mobile First - mulailah dengan gaya untuk perangkat mobile terlebih dahulu, lalu tambahkan media queries untuk layar yang lebih besar.
  • Hindari terlalu banyak breakpoint. Gunakan hanya yang diperlukan berdasarkan konten Anda.
  • Uji di perangkat nyata, bukan hanya di emulator browser.
  • Gunakan unit relatif seperti %, vw, vh, rem, dan em untuk ukuran yang lebih fleksibel.
