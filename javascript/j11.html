Pengenalan JavaScript

Apa itu JavaScript?

JavaScript adalah bahasa pemrograman yang paling banyak digunakan di dunia web. Ini adalah bahasa yang menjalankan kode di sisi klien (client-side) dan membuat halaman web menjadi interaktif.

1. Sejarah JavaScript

JavaScript dibuat oleh Brendan Eich pada tahun 1995 saat bekerja di Netscape. Awalnya bernama Mocha, kemudian LiveScript, dan akhirnya menjadi JavaScript.

2. Mengapa JavaScript Penting?

  • Membuat halaman web menjadi interaktif
  • Membuat animasi dan efek visual
  • Menghandle event (klik, ketukan tombol, dll)
  • Mengubah konten halaman secara dinamis
  • Membuat aplikasi web modern dan responsif

3. Cara Menggunakan JavaScript

HTML 
<script>
// Kode JavaScript di sini
</script>

4. Dasar-dasar JavaScript

  • var - Mendeklarasikan variabel (di JavaScript modern gunakan let atau const)
  • function - Membuat fungsi
  • if - Pengambilan keputusan
  • for dan while - Perulangan
  • addEventListener - Menangani event

5. Contoh Kode Sederhana

JavaScript 
// Menampilkan pesan di console
console.log("Hello, World!");

// Menampilkan pesan di halaman
alert("Selamat datang!");

6. JavaScript Modern

  • ES6+ (ECMAScript 6 dan seterusnya)
  • Arrow functions
  • Template literals
  • Promise dan async/await
  • Classes

7. JavaScript di Browser

JavaScript dapat berinteraksi dengan DOM (Document Object Model) untuk:

  • Mengubah konten halaman
  • Mengubah style elemen
  • Membuat animasi
  • Menghandle event pengguna
  • Membuat permintaan ke server (AJAX)

8. JavaScript di Server

Dengan Node.js, JavaScript dapat menjalankan kode di server, memungkinkan:

  • Membangun aplikasi server-side
  • Membuat API
  • Menangani database
  • Membuat aplikasi real-time

9. Tools dan Framework JavaScript

  • React - Untuk membangun UI komponen
  • Vue.js - Framework untuk aplikasi web modern
  • Angular - Framework full-stack
  • Node.js - Runtime JavaScript di server
  • Express.js - Framework untuk Node.js

10. Tips Belajar JavaScript

  • Mulai dengan dasar-dasar
  • Praktek dengan project kecil
  • Baca dokumentasi resmi
  • Gunakan tools pengembangan modern
  • Ikuti komunitas developer
