Introduction to HTML

Structure and Logic

The structure of XHTML documents are simple. Any command that is issued, must be closed to ensure correct execution. If commands are issued in sequence, then the last issued command must be terminated first.

HTML 
<b>this is bold
<i>this is bold and italic</i>
this is bold again</b>
this is plain text

There are no if.. then.. else.. statements in XHTML, therefore it is not a programming language. In fact, XHTML is a member of the SGML family. SGML stands for Standard Generalized Markup Language, which was created by IBM for their electronic typewriter. Based on this fact, we know that XHTML is a publishing language, with the main purpose of controlling page layout.
