Cara Menjalankan JavaScript

1. Menggunakan Browser Console

Browser console adalah cara termudah untuk menjalankan JavaScript. Berikut langkah-langkahnya:

  1. Tekan F12 atau klik kanan > Inspect
  2. Pilih tab "Console"
  3. Tulis kode JavaScript dan tekan Enter
Contoh di Console 
// Menampilkan teks
console.log("Halo, Dunia!");

// Operasi matematika
let hasil = 5 + 3;
console.log(hasil);

2. Menggunakan File HTML

Cara ini lebih umum digunakan untuk proyek yang lebih besar. Anda bisa menulis JavaScript:

  • Di dalam tag <script> di HTML
  • Di file JavaScript terpisah yang di-import

3. Inline Script

HTML dengan Script Inline 
<html>
<head>
  <title>Contoh JavaScript</title>
</head>
<body>
  <script>
    document.write("Halo, Dunia!");
  </script>
</body>
</html>

4. External Script

HTML dengan Script Eksternal 
<!-- index.html -->
<html>
<head>
  <title>Contoh JavaScript</title>
</head>
<body>
  <script src="script.js"></script>
</body>
</html>

<!-- script.js -->
console.log("Script berjalan!");

5. Menggunakan Event Handler

HTML dengan Event Handler 
<button onclick="tampilkanAlert()">Klik Saya</button>

<script>
function tampilkanAlert() {
  alert("Tombol diklik!");
}
</script>

6. Best Practices

  • Gunakan external script untuk kode yang lebih besar
  • Pisahkan JavaScript dari HTML untuk kode yang lebih bersih
  • Gunakan console.log() untuk debugging
  • Test kode di console sebelum mengintegrasikannya ke halaman
  • Gunakan komentar untuk menjelaskan kode yang kompleks

7. Tips Penggunaan Console

  • console.log() untuk menampilkan teks
  • console.warn() untuk peringatan
  • console.error() untuk error
  • console.table() untuk menampilkan data dalam tabel
  • console.time() dan console.timeEnd() untuk mengukur waktu
