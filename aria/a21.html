Peran (Roles) dalam ARIA

1. Pengenalan Peran ARIA

Peran (roles) dalam ARIA adalah atribut yang digunakan untuk memberikan informasi tentang tujuan dan fungsi elemen dalam halaman web. Peran membantu teknologi bantuan memahami struktur dan fungsi elemen, sehingga pengguna dengan keterbatasan dapat memahami konteks dan interaksi dengan elemen tersebut.

2. Jenis-jenis Peran ARIA

  • Landmark Roles: Menandai area penting dalam halaman web
  • Widget Roles: Mendefinisikan fungsi kontrol interaktif
  • Document Structure Roles: Menentukan struktur dokumen
  • Window Roles: Mengelompokkan konten terkait

3. Contoh Peran ARIA Umum

Contoh Penggunaan Peran 
<!-- Landmark Roles -->
<nav role="navigation">...</nav>
<main role="main">...</main>
<aside role="complementary">...</aside>

<!-- Widget Roles -->
<button role="button">Klik Saya</button>
<div role="menu">...</div>
<div role="dialog">...</div>

<!-- Document Structure -->
<header role="banner">...</header>
<footer role="contentinfo">...</footer>

4. Cara Menggunakan Peran ARIA

  • Gunakan peran HTML semantik terlebih dahulu
  • Gunakan ARIA roles hanya jika HTML semantik tidak mencukupi
  • Pastikan peran sesuai dengan konteks elemen
  • Gunakan satu peran per elemen
  • Uji dengan teknologi bantuan

5. Peran ARIA untuk Aplikasi Web

Beberapa peran ARIA yang penting untuk aplikasi web modern:

  • application: Menandai area aplikasi
  • tablist dan tab: Untuk navigasi tab
  • grid dan row: Untuk tabel interaktif
  • tree dan treeitem: Untuk struktur hierarkis
  • dialog: Untuk jendela modal

6. Best Practices Menggunakan Peran ARIA

  • Pahami fungsi peran sebelum menggunakan
  • Gunakan peran yang sesuai dengan konteks
  • Gunakan atribut ARIA yang mendukung
  • Uji dengan pembaca layar
  • Dokumentasikan penggunaan peran

7. Hubungan dengan HTML Semantik

Peran ARIA harus digunakan sebagai suplemen untuk HTML semantik, bukan pengganti. HTML semantik seperti <nav>, <main>, dan <header> sudah memiliki peran bawaan yang seharusnya tidak perlu ditambahkan secara manual.

8. Contoh Penggunaan Praktis

Contoh Menu Navigasi 
<nav role="navigation">
  <ul role="menubar">
    <li role="none">
      <a href="#" role="menuitem">Beranda</a>
    </li>
    <li role="none">
      <a href="#" role="menuitem">Produk</a>
    </li>
    <li role="none">
      <a href="#" role="menuitem">Kontak</a>
    </li>
  </ul>
</nav>

9. Tips Penggunaan Peran ARIA

  • Gunakan peran yang paling spesifik
  • Pahami hierarki peran
  • Gunakan dengan properti ARIA yang sesuai
  • Uji dengan berbagai teknologi bantuan
  • Ikuti perkembangan standar ARIA

10. Peran ARIA dan SEO

Peran ARIA tidak hanya membantu aksesibilitas, tetapi juga dapat meningkatkan SEO:

  • Membantu search engine memahami struktur situs
  • Meningkatkan relevansi konten
  • Membantu pengindeksan yang lebih baik
  • Meningkatkan pengalaman pengguna
  • Membantu konten dinamis diindeks
