Membuat Halaman Biodata Sederhana

Panduan Langkah demi Langkah

Mari buat halaman biodata pribadi yang menarik dengan HTML dan CSS dasar.

1. Struktur Dasar HTML

HTML 
<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Biodata Saya</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <div class="container">
    <header>
      <img src="foto.jpg" alt="Foto Profil">
      <h1>Nama Anda</h1>
      <p>Web Developer</p>
    </header>
    
    <main>
      <section>
        <h2>Tentang Saya</h2>
        <p>Deskripsi singkat tentang diri Anda.</p>
      </section>
      
      <section>
        <h2>Kontak</h2>
        <p>Email: email@contoh.com</p>
        <p>Telepon: +62 812-3456-7890</p>
      </section>
    </main>
    
    <footer>
      <p>© 2025 Biodata Saya</p>
    </footer>
  </div>
</body>
</html>

2. Menambahkan Gaya dengan CSS

CSS (style.css) 
body {
  font-family: Arial, sans-serif;
  line-height: 1.6;
  margin: 0;
  padding: 20px;
  background-color: #f4f4f4;
}

.container {
  max-width: 800px;
  margin: 0 auto;
  background: white;
  padding: 20px;
  border-radius: 8px;
  box-shadow: 0 0 10px rgba(0,0,0,0.1);
}

header {
  text-align: center;
  padding: 20px 0;
}

header img {
  width: 150px;
  height: 150px;
  border-radius: 50%;
  object-fit: cover;
  border: 3px solid #0d6efd;
}

h1 {
  color: #333;
  margin: 10px 0 5px;
}

h2 {
  color: #0d6efd;
  border-bottom: 2px solid #eee;
  padding-bottom: 5px;
}

section {
  margin: 20px 0;
  padding: 0 10px;
}

footer {
  text-align: center;
  margin-top: 30px;
  padding: 10px;
  color: #666;
  font-size: 0.9em;
}

/* Responsive Design */
@media (max-width: 600px) {
  .container {
    padding: 10px;
  }
  
  header img {
    width: 120px;
    height: 120px;
  }
}

}

}
