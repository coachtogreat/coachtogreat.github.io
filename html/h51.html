Daftar Berurutan (Ordered List) HTML

Memahami Tag <ol> dan <li> dalam HTML

Daftar berurutan (ordered list) digunakan untuk menampilkan daftar item yang memiliki urutan atau langkah-langkah. Setiap item dalam daftar secara otomatis diberi nomor oleh browser.

1. Dasar Penggunaan <ol>

Berikut adalah contoh dasar penggunaan tag <ol>:

HTML 
<ol>
  <li>Item pertama</li>
  <li>Item kedua</li>
  <li>Item ketiga</li>
</ol>
Hasil:
  1. Item pertama
  2. Item kedua
  3. Item ketiga

2. Atribut Utama <ol>

type

Menentukan jenis penomoran yang digunakan:

HTML 
<ol type="1">...</ol> <!-- Angka (default) -->
<ol type="A">...</ol> <!-- Huruf besar (A, B, C) -->
<ol type="a">...</ol> <!-- Huruf kecil (a, b, c) -->
<ol type="I">...</ol> <!-- Angka Romawi besar (I, II, III) -->
<ol type="i">...</ol> <!-- Angka Romawi kecil (i, ii, iii) -->
Contoh:
Angka (1, 2, 3):
  1. Pertama
  2. Kedua
Huruf Besar (A, B, C):
  1. Pertama
  2. Kedua
Angka Romawi (I, II, III):
  1. Pertama
  2. Kedua
start

Menentukan nomor awal untuk daftar:

HTML 
<ol start="5">
  <li>Item kelima</li>
  <li>Item keenam</li>
</ol>
Hasil:
  1. Item kelima
  2. Item keenam
reversed

Mengurutkan daftar secara terbalik (tidak memerlukan nilai):

HTML 
<ol reversed>
  <li>Item ketiga</li>
  <li>Item kedua</li>
  <li>Item pertama</li>
</ol>
Hasil:
  1. Item ketiga
  2. Item kedua
  3. Item pertama

3. Atribut <li>

value

Mengubah nomor urut untuk item tertentu:

HTML 
<ol>
  <li>Pertama</li>
  <li value="5">Lima</li>
  <li>Enam</li>
</ol>
Hasil:
  1. Pertama
  2. Lima
  3. Enam

4. Daftar Bersarang (Nested Lists)

Anda dapat membuat daftar di dalam daftar untuk membuat hierarki yang lebih kompleks:

HTML 
<ol>
  <li>Bab 1
    <ol type="a">
      <li>Sub-bab 1.1</li>
      <li>Sub-bab 1.2</li>
    </ol>
  </li>
  <li>Bab 2
    <ol type="i">
      <li>Sub-bab 2.1</li>
      <li>Sub-bab 2.2</li>
    </ol>
  </li>
</ol>
Hasil:
  1. Bab 1
    1. Sub-bab 1.1
    2. Sub-bab 1.2
  2. Bab 2
    1. Sub-bab 2.1
    2. Sub-bab 2.2

5. Menggabungkan dengan CSS

Anda menyesuaikan tampilan daftar dengan CSS:

HTML 
<style>
  .daftar-kustom {
    list-style-type: none;
    counter-reset: langkah;
    padding: 0;
  }
  .daftar-kustom li {
    counter-increment: langkah;
    margin-bottom: 10px;
    position: relative;
    padding-left: 40px;
  }
  .daftar-kustom li::before {
    content: counter(langkah);
    background-color: #007bff;
    color: white;
    font-weight: bold;
    border-radius: 50%;
    width: 25px;
    height: 25px;
    display: inline-flex;
    align-items: center;
    justify-content: center;
    position: absolute;
    left: 0;
    top: 0;
  }
</style>

<ol class="daftar-kustom">
  <li>Langkah pertama</li>
  <li>Langkah kedua</li>
  <li>Langkah ketiga</li>
</ol>
Hasil:
  1. Langkah pertama
  2. Langkah kedua
  3. Langkah ketiga

6. Praktik Terbaik

  • Gunakan daftar berurutan hanya ketika urutan item memang penting
  • Hindari menggunakan daftar hanya untuk tujuan indentasi
  • Gunakan CSS untuk mengubah gaya daftar daripada menggunakan atribut yang sudah tidak didukung seperti compact
  • Pastikan daftar tetap dapat dibaca dan dipahami tanpa CSS
  • Gunakan daftar bersarang dengan bijak untuk menghindari hierarki yang terlalu dalam
