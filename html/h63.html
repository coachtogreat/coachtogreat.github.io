Styling Tabel Dasar dengan CSS

Mempercantik Tabel HTML dengan CSS

CSS memberikan fleksibilitas tinggi dalam mengatur tampilan tabel HTML. Di halaman ini, kita akan mempelajari berbagai properti CSS yang dapat digunakan untuk mempercantik tabel.

Catatan: Sebelum mempelajari styling, pastikan Anda sudah memahami struktur dasar tabel HTML yang telah dibahas di halaman sebelumnya.

1. Dasar-dasar Styling Tabel

a. Border Tabel

Mengatur border pada tabel dan sel-selnya:

CSS 
/* Border untuk tabel */
table {
  border-collapse: collapse; /* Menggabungkan border yang bersebelahan */
  width: 100%;
  margin-bottom: 1rem;
  border: 1px solid #dee2e6;
}

/* Border untuk sel header dan data */
th, td {
  border: 1px solid #dee2e6;
  padding: 0.75rem;
  text-align: left;
}

/* Border khusus untuk header */
th {
  border-bottom: 2px solid #dee2e6;
  background-color: #f8f9fa;
  font-weight: bold;
}

Nama Usia Kota

b. Warna dan Latar Belakang

Menambahkan warna untuk meningkatkan keterbacaan:

CSS 
/* Warna baris ganjil-genap */
tr:nth-child(even) {
  background-color: #f8f9fa;
}

/* Efek hover pada baris */
tr:hover {
  background-color: #e7f5ff;
}

/* Warna header */
thead th {
  background-color: #0d6efd;
  color: white;
}

/* Warna footer */
tfoot td {
  background-color: #e9ecef;
  font-weight: bold;
}
c. Spasi dan Padding

Mengatur spasi dalam dan luar sel:

CSS 
/* Padding sel */
th, td {
  padding: 12px 15px; /* 12px atas-bawah, 15px kiri-kanan */
}

/* Spasi antar sel */
table {
  border-spacing: 0; /* Hanya bekerja jika border-collapse: separate */
}

/* Jarak antar baris */
tbody tr {
  margin-bottom: 5px; /* Tidak berlaku untuk display: table-row */
}

2. Tabel Responsif

CSS 
/* Container untuk tabel responsif */
.table-responsive {
  width: 100%;
  overflow-x: auto;
  -webkit-overflow-scrolling: touch; /* Untuk scrolling halus di iOS */
  margin-bottom: 1rem;
}

/* Media query untuk perangkat kecil */
@media screen and (max-width: 768px) {
  table {
    display: block;
  }
  
  thead, tbody, th, td, tr {
    display: block;
  }
  
  thead tr {
    position: absolute;
    top: -9999px;
    left: -9999px;
  }
  
  tr {
    margin-bottom: 1rem;
    border: 1px solid #dee2e6;
  }
  
  td {
    position: relative;
    padding-left: 50%;
    text-align: right;
    border-bottom: 1px solid #dee2e6;
  }
  
  td:before {
    content: attr(data-label);
    position: absolute;
    left: 15px;
    width: 45%;
    text-align: left;
    font-weight: bold;
  }
}
Penting: Pastikan untuk menambahkan atribut data-label pada setiap <td> yang berisi teks label untuk tampilan responsif.

3. Efek Visual Lanjutan

a. Striped Table
CSS 
/* Striped rows */
.striped tbody tr:nth-of-type(odd) {
  background-color: rgba(0, 0, 0, 0.02);
}

.striped tbody tr:hover {
  background-color: rgba(0, 0, 0, 0.05);
  transition: background-color 0.2s ease;
}
b. Bordered Table
CSS 
/* Tabel dengan border lengkap */
.bordered {
  border: 1px solid #dee2e6;
  border-radius: 4px;
  overflow: hidden;
}

.bordered th,
.bordered td {
  border: 1px solid #dee2e6;
}

.bordered thead th {
  border-bottom-width: 2px;
}
c. Hover Effect
CSS 
/* Efek hover pada sel */
.hover-cell td {
  transition: all 0.2s ease;
}

.hover-cell tbody tr:hover td {
  background-color: #e9ecef;
  transform: translateX(2px);
  box-shadow: 0 2px 5px rgba(0,0,0,0.1);
}

5. Praktik Terbaik

  • Gunakan border-collapse: collapse untuk tampilan border yang lebih rapi
  • Pastikan ada cukup kontras antara teks dan latar belakang
  • Gunakan padding yang cukup untuk keterbacaan yang baik
  • Pertimbangkan aksesibilitas dengan memastikan tabel dapat dinavigasi dengan keyboard
  • Uji tampilan di berbagai ukuran layar
  • Gunakan class yang deskriptif untuk gaya yang dapat digunakan kembali
  • Hindari penggunaan !important yang tidak perlu
  • Pertimbangkan untuk menggunakan CSS variables untuk warna dan ukuran yang konsisten
